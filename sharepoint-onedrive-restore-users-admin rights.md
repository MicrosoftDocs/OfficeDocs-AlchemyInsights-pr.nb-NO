<p><span data-ttu-id="ba77e-103"><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Dette skjer oftest når en bruker slettes og opprettes på nytt med samme brukerhovednavnet (UPN). A second scenario involves directory synchronization with an Active Directory organizational unit (OU).
