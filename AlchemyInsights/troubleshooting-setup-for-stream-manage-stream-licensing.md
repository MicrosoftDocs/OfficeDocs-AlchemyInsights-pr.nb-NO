# <a name="managing-microsoft-stream-licenses"></a>Administrere Microsoft stream-lisenser

**Viktig:** Du må være i rollen som leier administrator for å utføre operasjoner som er beskrevet i dette emnet.

Du må enten være global administrator eller administrator for bruker administrasjon. Hvis du vil ha mer informasjon, kan du se [om administrator roller](https://docs.microsoft.com/microsoft-365/admin/add-users/about-admin-roles).

Du kan tilordne [lisenser til bruker kontoer med Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
