# <a name="managing-microsoft-stream-licenses"></a>Administrere Microsoft Stream-lisenser

**Viktig:** Du må være i rollen Leieradministrator for å utføre operasjoner som er beskrevet i dette emnet.

Du må være enten global administrator eller administrator for brukeradministrasjon. Hvis du vil ha mer informasjon, kan du se [Om administratorroller](https://docs.microsoft.com/microsoft-365/admin/add-users/about-admin-roles).

Du kan tilordne [lisenser til brukerkontoer med Office 365 PowerShell](https://go.microsoft.com/fwlink/p/?linkid=850410).
