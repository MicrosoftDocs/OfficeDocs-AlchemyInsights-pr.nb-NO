# <a name="new-feature-get-feedback-about-spam-judgments"></a>Ny funksjon: få tilbakemelding om spam judgments

Som administrator, kan du nå enkelt rapportere mistenkelige e-postmeldinger, URL-adresser og innhold til Microsoft i [Explorer sendinger](https://protection.office.com/reportsubmission). Du kan også bruke den til å forstå hvorfor en lovlig melding ble flagget på feil måte. Hvis du vil ha mer informasjon, kan du se denne [kunngjøringen](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

Hvis du vil gjøre det easer, kan du tillate brukerne å rapportere direkte til Microsoft via den [rapportmelding-tillegg](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
