# <a name="run-microsoft-edge-in-kiosk-mode"></a>Kjøre Microsoft Edge i kioskmodus

Du kan konfigurere kioskmodus i Microsoft Edge-versjon 77 og nyere, i tillegg til Microsoft Edge Legacy-versjonene 45 og tidligere.

Du kan konfigurere at enheter som kjører på Windows 10, skal kjøre i [kioskmodus](https://go.microsoft.com/fwlink/?linkid=2133659)med én app eller flere apper.

Hvis du vil ha mer informasjon om hvordan du konfigurerer kioskmodus i den nyeste versjonen av Microsoft Edge Stable-kanalen og i Microsoft Edge Legacy, kan du se:

- [Microsoft Edge med tilordnet tilgang](https://docs.microsoft.com/deployedge/microsoft-edge-configure-kiosk-mode#microsoft-edge-with-assigned-access)
- [Kommandolinjeparameteren Microsoft Edge --kiosk](https://answers.microsoft.com/microsoftedge/forum/msedge_open-msedge_win10/access-microsoft-edge-using-command-line/03a4add6-9ca4-4fbb-a183-aaa763a0ab76)
- [Støtte for Microsoft Edge Legacy-kioskmodus](https://blogs.windows.com/msedgedev/2021/02/05/what-you-need-to-know-about-kiosk-mode-when-support-for-microsoft-edge-legacy-ends/)
