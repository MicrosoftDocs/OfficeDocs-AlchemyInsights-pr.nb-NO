# <a name="email-forwarding-report"></a>Rapport om videresending av e-post

Hvis du vil se en rapport om hvor mange meldinger som blir automatisk videresendt i organisasjonen din, kan du se [Innsikt om e-postflyt](https://support.office.com/article/beb6acaa-6016-4d54-ba7e-3d6d035e2b46) i [Office 365 sikkerhets- og samsvarssenter](https://protection.office.com/#/homepage).
  
Merk: Hvis dette er første gang du logger på sikkerhets- og samsvarssenteret, må du kanskje oppdatere nettleseren etter noen få minutter for å se alle de tilgjengelige alternativene.
