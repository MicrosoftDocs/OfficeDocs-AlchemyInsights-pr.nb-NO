# <a name="fix-windows-update-errors"></a>Løse Windows Update-feil

Hvis du får en feilkode mens du laster ned og installerer Windows Update, kan **feil søking for oppdatering** hjelpe til med å løse problemet.

- [Last ned oppdaterings feil søking for Windows 10](https://support.microsoft.com/help/4027322/windows-update-troubleshooter)
- Kjør feil søking. Når det er ferdig, er det best å starte enheten på nytt. Se etter nye oppdateringer ved å gå til [Start > innstillinger > oppdater & sikkerhets > Windows Update](ms-settings:windowsupdate).

Hvis disse fortsatt ikke fungerer, kan du se [vanlige spørsmål](https://support.microsoft.com/help/12373/windows-update-faq) for å få mer informasjon.
