# <a name="desktop-app-for-sharepoint"></a>Desktop app for SharePoint

Organisasjonen er angitt i den Private forhåndsvisningsprogrammet for Desktop-App for SharePoint.

Microsoft støtter ikke for øyeblikket Desktop-App for SharePoint (Private forhåndsvisning). Bruk de innrullerte Microsoft Private forhåndsvisning-partnerne for å få hjelp med PC-App for SharePoint.
