# <a name="desktop-app-for-sharepoint"></a>Skrive bords app for SharePoint

Organisasjonen skrives inn i forhånds visnings programmet for privat data maskin-appen for SharePoint.

Microsoft støtter for øyeblikket ikke skrive bords programmet for SharePoint (privat forhånds visning). Bruk de oppførte Microsoft private forhånds visnings partnerne til å få hjelp med skrive bords programmet for SharePoint.
