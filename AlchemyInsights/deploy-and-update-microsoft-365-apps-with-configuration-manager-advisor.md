# <a name="deploy-and-update-microsoft-365-apps-with-configuration-manager-advisor"></a>Distribuere og oppdatere Microsoft 365-apper med Configuration Manager-rådgiver

For organisasjoner som bruker Configuration Manager, kan du bruke Distribuer og oppdater [Microsoft 365-apper](https://go.microsoft.com/fwlink/?linkid=2146549) med Configuration Manager-rådgiveren til å generere et skript som automatisk konfigurerer distribusjonen av Microsoft 365-apper ved hjelp av anbefalte fremgangsmåter som anbefales av FastTrack-teknikere.

Bruk denne veiledningen til å bygge distribusjonsgrupper, tilpasse Office-apper og -funksjoner, konfigurere dynamiske eller magre installasjoner og deretter kjøre skriptet for å opprette programmer, regler for automatisk distribusjon og enhetssamlinger du trenger for å målrette distribusjonen.
