# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Løs "Godkjenning mislyktes" feil i Microsoft Flow

Hvis du får feilmeldingen "Godkjenning mislyktes" når du prøver å logge på Microsoft Flow, er det fordi Flow-tjenesten er deaktivert i Azure Active Directory. Hvis du vil ha mer informasjon og trinn for å rette opp feilen, kan du se [Du kan ikke logge på Microsoft Flow, og du får feilmeldingen "Godkjenning mislyktes"](https://support.microsoft.com/help/4316891).
