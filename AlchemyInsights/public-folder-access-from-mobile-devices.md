# <a name="public-folder-access-from-mobile-devices"></a>Tilgang til felles mapper fra mobile enheter

Tilgang til felles mapper fungerer ikke med mobile enheter, du kan bruke OWA (Outlook på nettet) eller Outlook for Windows-skrivebordet eller Outlook for Mac-klienter til å få tilgang til felles mapper.

Hvis du vil ha funksjonalitet som ligner på felles mapper og som har innhold som er tilgjengelig på mobile enheter, kan du se [lære om Microsoft 365-grupper](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) for å få et bedre alternativ.
