# <a name="changes-to-external-user-access"></a>Endringer i ekstern bruker tilgang

Starter 23 mars, 2018, eksterne brukere vil ikke lenger kunne få tilgang til innhold som er delt med alle, alle godkjente brukere eller alle skjemaer-brukere, selv om de er invitert eller lagt til autoriserte grupper. Eksterne brukere kan bare få tilgang til innhold som er delt med brukeren eller grupper som brukeren tilhører. Hvis du vil beholde de eksisterende innstillingene, kan du se:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
