# <a name="changes-to-external-user-access"></a>Endringer i ekstern bruker tilgang

Hvis du starter mars 23, 2018, vil ikke eksterne brukere lenger få tilgang til innhold som er delt med alle, alle godkjente brukere eller alle skjema brukere, selv om de har blitt invitert eller lagt til i autoriserte grupper. Eksterne brukere vil bare kunne få tilgang til innhold som er delt med brukeren, eller med grupper som brukeren tilhører. Hvis du ønsker å beholde eksisterende innstillinger, kan du se: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
