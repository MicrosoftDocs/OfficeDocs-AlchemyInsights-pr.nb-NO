# <a name="changes-to-external-user-access"></a>Endringer i ekstern brukertilgang

Starter 23 mars 2018 lenger eksterne brukere ikke kunne få tilgang til innhold som er delt med alle, alle, godkjente brukere eller alle brukere av skjemaer, selv om de har invitert eller lagt til autoriserte grupper. Eksterne brukere bare kan få tilgang til innhold som er delt med brukeren eller gruppene som brukeren tilhører. Hvis du vil beholde de eksisterende innstillingene, kan du se:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
