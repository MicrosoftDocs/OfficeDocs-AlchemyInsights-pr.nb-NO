# <a name="changes-to-external-user-access"></a>Endringer i ekstern brukertilgang

Fra og med 23. mars 2018 vil eksterne brukere ikke lenger kunne få tilgang til innhold som er delt med alle, alle godkjente brukere eller alle skjemabrukere, selv om de er invitert eller lagt til i autoriserte grupper. Eksterne brukere får bare tilgang til innhold som er delt med brukeren eller med grupper som brukeren tilhører. Hvis du vil beholde eksisterende innstillinger, kan du se:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
