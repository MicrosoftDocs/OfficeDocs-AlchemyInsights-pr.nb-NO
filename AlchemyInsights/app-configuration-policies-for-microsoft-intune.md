# <a name="app-configuration-policies-for-microsoft-intune"></a>Policyer for appkonfigurasjon for Microsoft Intune

Appkonfigurasjonspolicyer i Microsoft Intune gir konfigurasjonsinnstillinger for iOS- eller Android-programmer. Disse konfigurasjonsinnstillingene gjør at en app kan tilpasses. Støtte for konfigurasjonspolicy er begrenset av operativsystemversjoner og programmer.

Hvis du vil ha mer informasjon, kan du se [Policyer for appkonfigurasjon for Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
