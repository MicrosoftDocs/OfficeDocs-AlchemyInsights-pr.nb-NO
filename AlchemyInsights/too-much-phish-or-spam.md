# <a name="are-you-receiving-too-much-phish-or-spam-in-your-mailbox"></a>Du mottar for mye phish eller spam i postboksen?

Som en Microsoft 365 admin er den beste måten å finne ut hvorfor en phishing eller spam-melding ble levert til å sende utvalget til Microsoft i [Explorer sendinger](https://protection.office.com/reportsubmission). Du får umiddelbar tilbakemelding som er relatert til meldingsfiltrering og hvorfor det er problemer.

Hvis du vil gjøre det easer, kan du tillate brukerne å rapportere direkte til Microsoft via den [rapportmelding-tillegg](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
