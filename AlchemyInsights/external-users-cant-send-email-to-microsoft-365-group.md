# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Eksterne brukere kan ikke sende e-post til Microsoft 365-gruppen

Bruk disse trinnene til å tillate eksterne brukere å sende microsoft 365-gruppen via e-post:

1. Logg på [administrasjonssenteret](https://admin.microsoft.com/)  >  **Grupper**  >  **grupper**.
2. Velg gruppen du vil administrere for å åpne Innstillinger-ruten.
3. Velg **Settings** **Tillat eksterne avsendere å sende denne gruppen via e-post**.
4. Velg **Lagre**.</br>
    Tillat opptil en time før innstillingene gjenspeiles.
