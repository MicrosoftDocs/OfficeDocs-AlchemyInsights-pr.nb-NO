# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Støttet versjon av overførings verktøyet for SharePoint



Overførings verktøyet i SharePoint støtter for øyeblikket ikke overføringer fra SharePoint 2013. Versjon 2010 og 2016 støttes ikke på dette tidspunktet.
  
Hvis du vil ha mer informasjon, kan du se [innføring i overførings verktøyet for SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
