# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Støttet versjon av SharePoint Migration Tool



SharePoint Migration Tool støtter for øyeblikket bare overføringer fra SharePoint 2013. Versjon 2010 og 2016 støttes ikke for øyeblikket.
  
Hvis du vil ha mer informasjon, kan du se [Innføring i Overføringsverktøyet for SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
