# <a name="onedrive-cant-add-your-folder-right-now"></a>OneDrive kan ikke legge til mappen akkurat nå

Feilen «Beklager, OneDrive kan ikke legge til mappen akkurat nå» kan oppstå  hvis  du har biblioteker med kolonnene Utsjekking eller  Validering, eller når Sikkerhet for kladdeelement er satt til Bare brukere som kan redigere eller Bare brukere som kan godkjenne elementer i bibliotekets versjon Innstillinger.   

Hvis du vil ha instruksjoner for hvordan du fjerner innstillinger for synkroniseringsblokkering, kan du se Begrensninger og begrensninger i OneDrive [og SharePoint](https://support.microsoft.com/office/64883a5d-228e-48f5-b3d2-eb39e07630fa).
