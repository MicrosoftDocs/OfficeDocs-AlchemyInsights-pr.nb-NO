# <a name="change-the-domain-for-microsoft-365-group"></a>Endre domenet for Microsoft 365 Group

Hvis du vil endre domenet for en Microsoft 365-gruppe under oppretting, følger du fremgangsmåten i [Velg domenet som skal brukes når du oppretter Microsoft 365-grupper](https://docs.microsoft.com/office365/admin/create-groups/choose-domain-to-create-groups).
  
Når Microsoft 365-gruppen allerede er opprettet, er det for øyeblikket ingen måte å endre domenet på. Den eneste måten er å opprette en ny Microsoft 365-gruppe og slette den gamle om nødvendig (flere e-postadresser kan legges til via [Administrasjonssenter for Exchange](https://outlook.office365.com/ecp.aspx)).
