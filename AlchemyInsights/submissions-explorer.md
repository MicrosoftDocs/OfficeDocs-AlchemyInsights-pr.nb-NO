# <a name="new-feature-to-help-diagnose-eop-and-atp"></a>Ny funksjon for å diagnostisere EOP og ATP

Som administrator kan du nå enkelt rapportere mistenkelige e-postmeldinger, URL-adresser og innhold til Microsoft i [sendinger SUT forsker](https://protection.office.com/reportsubmission). Du kan til og med bruke det til å forstå hvorfor en legitim melding ble flagget på feil måte. Hvis du vil ha mer informasjon, kan du se denne [kunngjøringen](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).
