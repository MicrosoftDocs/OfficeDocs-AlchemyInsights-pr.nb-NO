# <a name="troubleshoot-error-code-aadsts50011"></a>Feilsøke feilkode AADSTS50011

Hvis du vil løse AADSTS50011-feil, utfører du det anbefalte trinnet som er beskrevet nedenfor.

**AADSTS50011**: InvalidReplyTo – Svaradressen mangler, er feilkonfigurert eller samsvarer ikke med svaradressene som er konfigurert for appen.

Som en løsning sikrer du at du legger til denne manglende svaradressen i Azure Active Directory-appen (AD), eller at noen med tillatelse til å administrere programmet i AD gjør dette for deg. Hvis du vil ha mer informasjon, kan du se feilsøkingsartikkelen for feil [AADSTS50011.](https://docs.microsoft.com/troubleshoot/azure/active-directory/error-code-aadsts50011-reply-url-mismatch)
