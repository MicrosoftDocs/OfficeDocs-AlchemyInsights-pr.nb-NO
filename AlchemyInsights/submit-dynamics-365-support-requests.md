# <a name="submit-dynamics-365-support-requests"></a>Sende forespørsler om Dynamics 365-støtte

Hvis du er en **Kundeengasjement for Microsoft Dynamics** -kunde kan du sende inn din billett gjennom den nye [Power plattform administrasjonssenteret](https://admin.powerplatform.microsoft.com/?ref=officemodern).
  
Den nye portalen gir en mer direkte bane til kundestøtte som er optimalisert for Dynamics kundeengasjement og andre Power plattform-produkter.
