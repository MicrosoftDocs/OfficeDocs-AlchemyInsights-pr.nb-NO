# <a name="global-reader"></a>Global leser

Brukere i denne rollen kan lese innstillinger og administrativ informasjon på tvers av Microsoft 365-tjenester, men kan ikke utføre administrasjonshandlinger. Global leser er det skrivebeskyttede motstykket til global administrator.
Administrasjonssenteret for OneDrive støtter ikke den globale leserrollen.

Hvis du vil lære mer om global leser og gjeldende begrensninger, kan du se [Global Reader](https://docs.microsoft.com/azure/active-directory/users-groups-roles/directory-assign-admin-roles#global-reader).
