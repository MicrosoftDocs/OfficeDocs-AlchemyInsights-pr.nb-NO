# <a name="plan-your-passwordless-deployment"></a>Planlegge distribusjon uten passord

Oppgrader til en alternativ pålogging ved hjelp av en godkjenningsmetode uten passord. Bekreftelse med to faktorer, telefon pålogging eller kodegeneratorer kan bidra til å øke sikkerheten og gjøre det vanskeligere for andre å bryte seg inn på kontoen din. 

Velg fra Windows Hello for Business, Microsoft Authentication-appen eller FIDO2-sikkerhetsnøkler. Hvis du vil finne ut hvilken metode som er best og hvordan du distribuerer dem på enheten, kan du bruke [distribusjonsveiledningen Uten passord](https://admin.microsoft.com/adminportal/home?#/modernonboarding/passwordlesssetup).
