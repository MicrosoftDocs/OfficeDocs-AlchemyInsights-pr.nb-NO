# <a name="get-help-with-windows-10-desktop"></a>Få hjelp med Windows 10 Desktop

- Hvis du vil opprette en snarvei på skrivebordet, kan du se [Feste apper og mapper til skrivebordet eller oppgavelinjen](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- Hvis du vil gjøre ikonene eller teksten større i Windows 10, kan du [se Endre størrelsen på skrivebordsikonene](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211).

- Hvis du vil vise skrivebordsikoner i Windows 10, kan du [se Vise skrivebordsikoner i Windows 10](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20).
