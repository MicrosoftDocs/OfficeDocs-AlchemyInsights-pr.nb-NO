# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>Synkronisere SharePoint-filer med den nye synkroniseringsklienten for OneDrive

Kommandoen Åpne med Utforsker åpner en lokal forekomst av Windows Utforsker som viser mappestrukturen på serveren som er vert for SharePoint-området. Vi anbefaler imidlertid at du [synkroniserer SharePoint-filer med den nye synkroniseringsklienten for OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88)</a> som gir tilgang til [filer ved behov](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e), fordi den gir lokal tilgang til filene og tilbyr den beste ytelsen.


Hvis du valgte å bruke utforskervisningen i stedet for å bruke den nye synkroniseringsklienten, må du passe på at du følger trinnene og gode fremgangsmåter i artiklene nedenfor.

- [Slik bruker du kommandoen Åpne med Utforsker til å feilsøke problemer i SharePoint Online](https://support.office.com/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4)

- [Kopiere eller flytte bibliotekfiler ved å bruke Åpne med Explorer](https://support.office.com/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2)

Obs! Åpne med Utforsker-knappen vises ikke i den nye bibliotekopplevelsen Klikk på Vis-rullegardinmenyen oppe til høyre (navnet på rullegardinmenyen endres basert på gjeldende visning), og klikk på Vis i Filutforsker.

 SharePoint Åpne med Explorer bruker ActiveX-kontroller, så det støttes bare i Internet Explorer 10 eller 11. Åpne med Explorer virker ikke i Windows med Microsoft Edge, Google Chrome, Mozilla Firefox eller på Mac-plattformen. På grunn av dette kan det hende at alternativet Explorer-visning er nedtonet.

- [Hvorfor er SharePoint-båndknapper utilgjengelige eller nedtonet](https://support.office.com/article/Why-SharePoint-ribbon-buttons-are-unavailable-48b0939a-2efb-4e79-b5e8-b2c4cb5d04ca)
