# <a name="sync-sharepoint-files-with-the-new-onedrive-sync-client"></a>Synkronisere SharePoint-filer med den nye synkroniseringsklienten for OneDrive

<p>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Kommandoen Åpne med Utforsker åpner en lokal forekomst av Windows Utforsker som viser mappestrukturen på serveren som er vert for SharePoint-området. Det sagt anbefaler vi <a href="https://support.office.com/nb-NO/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88">å synkronisere SharePoint-filer med den nye synkroniseringsklienten for OneDrive</a>, som tilbyr <a href="https://support.office.com/nb-NO/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e">filer ved behov</a>, fordi det gir lokal tilgang til filene dine og gir den beste ytelsen</span></p> <p><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Hvis du valgte å bruke utforskervisningen i stedet for å bruke den nye synkroniseringsklienten, må du passe på at du følger trinnene og gode fremgangsmåter i artiklene nedenfor.</span></p> <ul> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/nb-NO/article/How-to-use-the-Open-with-Explorer-command-to-troubleshoot-issues-in-SharePoint-Online-87155331-0c92-4224-a4c1-da5c21c4ade4">Slik bruker du kommandoen «Åpne med Utforsker» til å feilsøke problemer i SharePoint Online</a></span></li> <li>
  <span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">
  <a href="https://support.office.com/nb-NO/article/copy-or-move-library-files-by-using-open-with-explorer-aaee7bfb-e2a1-42ee-8fc0-bcc0754f04d2">Kopiere eller flytte bibliotekfiler ved å bruke Åpne med Explorer</a></span></li> </ul> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Merk:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> Åpne med Utforsker-knappen vises ikke i den nye bibliotekopplevelsen. Klikk på <strong>Vis</strong>-rullegardinmenyen oppe til høyre (navnet på rullegardinmenyen endres basert på gjeldende visning), og klikk på <strong>Vis</strong> i Filutforsker.</span></p> <p><strong style="mso-bidi-font-weight: normal;"><u><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;">Merk:</span></u></strong><span style="mso-bidi-font-family: Calibri; mso-bidi-theme-font: minor-latin;"> SharePoint Åpne med utforsker bruker <strong>ActiveX-kontroller</strong>, så funksjonen støttes bare i Internet Explorer 10 eller 11. Åpne med Utforsker fungerer ikke i Windows med Microsoft Edge, Google Chrome, Mozilla Firefox eller på Mac-plattformen. Av denne grunn kan &ldquo;Utforsker-visning&rdquo;-alternativet være nedtonet.</span></p> <ul> <li>
