# <a name="microsoft-search-setup-guide"></a>Installasjonsveiledning for Microsoft Søk

Bruk [installasjonsveiledningen for Microsoft Search](https://go.microsoft.com/fwlink/?linkid=2153798) til å konfigurere og konfigurere Microsoft Søk. Enten du vil prøve den ut til en gruppe brukere eller rulle det ut til alle i organisasjonen. Du vil tilordne søkeadministratorer og søkeredigering og deretter tilpasse søkeopplevelsen for brukerne med svar og flere alternativer, som å legge til Bing-utvidelsen i Chrome eller angi Bing som standard søkemotor.
