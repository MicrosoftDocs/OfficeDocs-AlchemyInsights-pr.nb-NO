# <a name="submit-powerapps-or-microsoft-flow-support-requests"></a>Sende powerapps- eller Microsoft Flow-støtteforespørsler

Hvis du er en **PowerApps-** eller **Microsoft Flow-kunde,** vennligst send inn billetten din via det nye [administrasjonssenteret for Power-plattformen](https://admin.powerplatform.microsoft.com/support?newTicket&product=15819). Den nye portalen gir en mer direkte bane til assistert støtte som er optimalisert for PowerApps og andre Power Platform-produkter.

[Sende forespørsel om Støtte for Microsoft Flow](https://admin.powerplatform.microsoft.com/support?newTicket&product=Flow)
