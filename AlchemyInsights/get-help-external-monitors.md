# <a name="get-help-with-external-monitors"></a>Få hjelp med eksterne skjermer

Du kan enkelt åpne prosjektinnstillingene ved å trykke følgende taster samtidig:

- Windows-tast (Startknapp for Windows-logo) + P
- Velg fra  **Skjermbildet Dupliser,** Utvid eller **Andre,** avhengig av innstillingene dine.

Hvis du har problemer med de eksterne skjermene, kan du prøve disse feilsøkingstrinnene:

- Koble kabelen som kobler PC-en til skjermen, og koble den til igjen.
- Koble alle ikke-essensielle enheter fra PC-en (for eksempel adaptere eller dokkingstasjoner).

Les mer om Microsoft Kundestøtte:

- [Konfigurere to skjermer på Windows 10](https://support.microsoft.com/windows/set-up-dual-monitors-on-windows-10-3d5c15dc-cc63-d850-aeb6-b41778147554)
- [Slik bruker du flere skjermer i Windows 10](https://support.microsoft.com/windows/how-to-use-multiple-monitors-in-windows-10-329c6962-5a4d-b481-7baa-bec9671f728a)
