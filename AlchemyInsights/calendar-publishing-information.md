# <a name="calendar-publishing-information"></a>Publiseringsinformasjon for kalender

- Administratorer for Microsoft 365 kan aktivere eller deaktivere kalenderpublisering og kalenderdeling for en organisasjon. Hvis du vil ha mer informasjon, se [aktivere kalenderpublisering slik at folk kan dele informasjon](https://support.office.com/article/EB432E21-AAF0-466B-BF85-CEFEC0C7C4FC) og [dele kalendere med eksterne brukere](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).

- Outlook-brukere kan dele sine egne kalendere. Hvis du vil ha mer informasjon, kan du se [dele en Outlook-kalender med andre personer](https://support.office.com/article/353ed2c1-3ec5-449d-8c73-6931a0adab88).
