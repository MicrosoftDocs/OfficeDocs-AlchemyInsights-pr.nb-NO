# <a name="enable-calendar-sharing-using-the-microsoft-365-admin-center"></a>Aktivere kalender deling ved hjelp av administrasjons senteret for Microsoft 365

1. Gå til **Innstillinger**for org i administrasjons senteret   >   **Org Settings**.
2. Velg **Kalender**på **tjenester** -fanen.
3. På  **Kalenders**  IDen som åpnes, velger du om du vil la brukerne dele kalenderne sine med personer utenfor organisasjonen som har Microsoft 365 eller Exchange.
4. Velg om du vil tillate anonyme brukere (brukere uten påloggings informasjon) for å få tilgang til kalendere via en e-postinvitasjon.
5. Velg hvilken type kalender informasjon som skal være tilgjengelig for brukere. Du kan tillate all informasjon, eller bare begrense til klokkeslett, emne og sted.

Mer informasjon: [dele kalendere med eksterne brukere](https://docs.microsoft.com/microsoft-365/admin/manage/share-calendars-with-external-users)
