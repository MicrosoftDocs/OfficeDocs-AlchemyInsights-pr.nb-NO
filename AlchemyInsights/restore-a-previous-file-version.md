# <a name="restore-a-previous-file-version"></a>Gjenopprette en tidligere versjon

Du kan se og gjenopprette eldre versjoner av filene i OneDrive eller SharePoint med online versjonslogg. Versjonslogg fungerer med alle filtyper, inkludert PDF-filer, CAD-filer, bilder og videoer. Å gjenopprette en tidligere versjon av filen:
  
1. Åpne SharePoint-dokumentbibliotek eller OneDrive plassering.
    
2. Klikk til venstre for å merke den, og klikk deretter **versjonsloggen**. Hvis du ikke ser versjonslogg på kommandolinjen, klikker du mer ( **...** ). 
    
3. Klikk versjonen du vil vise, eller klikk flere ( **...** ) til høyre for en versjon og klikker **Gjenopprett**.
