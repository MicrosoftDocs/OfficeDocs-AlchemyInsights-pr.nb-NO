# <a name="restore-a-previous-file-version"></a>Gjenopprette en tidligere filversjon

Med online versjonslogg, kan du se og gjenopprette eldre versjoner av filene dine i OneDrive eller SharePoint. Versjonsloggen fungerer med alle filtyper, inkludert PDF-er, CAD-filer, bilder og videoer. Slik gjenoppretter du en tidligere filversjon:
  
1. Åpne SharePoint-dokumentbibliotek eller OneDrive-plasseringen.
    
2. Klikk til venstre for filen for å merke den, og klikk deretter **Versjonslogg**. Hvis du ikke ser versjonslogg på kommandolinjen, klikker du mer ( **...** ). 
    
3. Klikk versjonen du vil vise, eller klikk mer (.. **.** ) til høyre for en versjon, og klikk deretter **Gjenopprett**.
