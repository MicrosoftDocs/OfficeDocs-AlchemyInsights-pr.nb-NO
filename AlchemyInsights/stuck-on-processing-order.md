# <a name="stuck-on-processing-order"></a>Ordren sitter fast

Hvis du har valgt å oppgradere Microsoft 365-abonnementet, men bestillingen sitter fast på «behandler bestillingen», kan du se [Instrumentpanel for tjenestestatus](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/servicehealth) innlegg **SP207374**. Hvis prosessen med å oppgradere leieren din krever klargjøring, noe som kan være forsinket. Vi forventer å fullføre prosessen innen 24 timer. Vi har imidlertid sett sjeldne forekomster der disse forespørslene tar opptil 48 timer. Åpne en serviceforespørsel hvis ordrebehandling tar lengre tid enn 48 timer.
