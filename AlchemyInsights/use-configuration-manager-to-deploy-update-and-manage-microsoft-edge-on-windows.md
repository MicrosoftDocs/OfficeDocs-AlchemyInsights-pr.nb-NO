# <a name="use-configuration-manager-to-deploy-update-and-manage-microsoft-edge-on-windows"></a>Bruke konfigurasjons behandling til å distribuere, oppdatere og administrere Microsoft Edge på Windows

I henhold til Configuration Manager versjon 1910 kan du distribuere Microsoft Edge (versjon 77 og nyere) gjennom et PowerShell-skript. Skriptet deaktiverer automatiske oppdateringer, og dette lar deg administrere oppdateringene gjennom konfigurasjons behandling. Hvis du vil lære mer om dette og andre fordeler ved å bruke Configuration Manager, kan du se [Microsoft Edge Management](https://docs.microsoft.com/mem/configmgr/apps/deploy-use/deploy-edge?).
