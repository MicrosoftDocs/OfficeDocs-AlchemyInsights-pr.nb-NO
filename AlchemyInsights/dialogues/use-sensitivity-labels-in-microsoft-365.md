# <a name="use-sensitivity-labels-in-microsoft-365"></a>Bruk følsomhetsetiketter i Microsoft 365

Du kan bruke følsomhetsetiketter til å:
- Bruk beskyttelsesinnstillinger som kryptering eller vannmerker på merket innhold.
- Beskytt innhold i Office-apper på tvers av forskjellige plattformer og enheter.
- Beskytt innhold i tredjepartsapper og -tjenester ved hjelp av Microsoft Cloud App Security.
- Utvid følsomhetsetiketter til apper og tjenester fra tredjeparter.
- Klassifiser innhold uten å bruke beskyttelsesinnstillinger.

Hvis du vil ha mer informasjon, kan [du se Lær om følsomhetsetiketter.](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)
