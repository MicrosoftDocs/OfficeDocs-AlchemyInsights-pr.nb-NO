# <a name="configure-message-encryption-for-a-hybrid-environment"></a>Konfigurere meldingskryptering for et hybridmiljø

I hybride Exchange-miljøer kan lokale brukere bare sende kryptert e-post ved hjelp av Office-meldingskryptering (OME) hvis e-post rutes via Exchange Online.

Utfør følgende trinn for å kryptere e-postmeldinger ved hjelp av OME:

1. Bruk veiviseren [for hybridkonfigurasjon til](https://docs.microsoft.com/Exchange/hybrid-configuration-wizard) å konfigurere hybridmiljøet. Det kreves ingen spesielle trinn for å konfigurere kryptering.
2. [Konfigurer e-postflytreglene for kryptering](https://docs.microsoft.com/microsoft-365/compliance/define-mail-flow-rules-to-encrypt-email) på samme måte som vanlig.
