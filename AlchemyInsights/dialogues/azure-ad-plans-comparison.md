# <a name="azure-active-directory-plans-comparison"></a>Sammenligning av Azure Active Directory-planer

Azure Active Directory leveres i fire versjoner: Gratis, Office 365-apper, Premium P1 og Premium P2.

Gratisutgaven er inkludert i et Azure-abonnement. Premium-versjonene er tilgjengelige gjennom Microsoft-representanten din, [Open Volume License Program](https://go.microsoft.com/fwlink/?linkid=2110873)og programmer for [skyløsninger.](https://go.microsoft.com/fwlink/?LinkId=614968&clcid=0x409)

Azure- og Office 365-abonnenter kan også kjøpe Azure Active Directory Premium P1 og P2 på nettet. Hvis du vil ha mer informasjon, kan du [se funksjoner og priser for Azure Active Directory.](https://go.microsoft.com/fwlink/?linkid=2081447)
