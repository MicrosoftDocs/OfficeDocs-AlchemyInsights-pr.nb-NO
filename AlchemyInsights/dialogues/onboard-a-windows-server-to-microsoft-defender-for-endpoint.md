# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Sette Windows-serveren på Microsoft Defender for endepunkt

For Windows Server-operativsystemet gir Microsoft Defender for endepunkt avanserte funksjoner for å oppdage og undersøke angrep. Disse funksjonene gis sømløst via Microsoft Defender Sikkerhetssenter-konsollen.

Defender for endepunkt støtter følgende versjoner av Windows Server:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) versjon 1803 og nyere
- Windows Server 2019 og nyere
- Windows Server 2019 core edition

Hvis du vil ha mer informasjon, kan du se [Onboard Windows-servere til Microsoft Defender for Endpoint service.](https://go.microsoft.com/fwlink/?linkid=2143627)
