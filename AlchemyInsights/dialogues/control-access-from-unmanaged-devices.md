# <a name="control-the-access-from-unmanaged-devices"></a>Kontrollere tilgangen fra ikke-administrerede enheter

Som SharePoint- eller global administrator i Microsoft 365 kan du blokkere eller begrense tilgangen til SharePoint- og OneDrive-innhold fra ikke-administrerte enheter. Hvis du vil ha mer informasjon, kan du se Kontrollere [tilgang fra ikke-administrerede enheter.](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices)
