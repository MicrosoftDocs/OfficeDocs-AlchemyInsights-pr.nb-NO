# <a name="publishing-infrastructure-with-a-communication-site"></a>Infrastruktur for publisering med et kommunikasjons nettsted


Aktivering av funksjonen for publiserings infrastruktur for SharePoint Server **støttes ikke** med kommunikasjons nett steder. 
  
Hvis du vil ha mer informasjon, kan du se [flytte fra publiserings områder til kommunikasjons nett steder](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
