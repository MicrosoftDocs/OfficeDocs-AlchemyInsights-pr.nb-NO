# <a name="publishing-infrastructure-with-a-communication-site"></a>Publisere infrastruktur med et kommunikasjonsområde


Aktivering av funksjonen infrastruktur for publisering av SharePoint Server **støttes ikke** med kommunikasjonsområder. 
  
Hvis du vil ha mer informasjon, kan du se [Flytte fra publiseringsområder til kommunikasjonsområder](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
