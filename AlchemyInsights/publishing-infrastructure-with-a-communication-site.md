# <a name="publishing-infrastructure-with-a-communication-site"></a>Publisere infrastruktur med et område for kommunikasjon


Aktivering av SharePoint Server publisering infrastruktur-funksjonen **støttes ikke** med kommunikasjons områder. 
  
Hvis du vil ha mer informasjon, kan [du se flytte fra publiseringsområder til kommunikasjons områder](https://docs.microsoft.com/sharepoint/publishing-sites-classic-to-modern-experience).
