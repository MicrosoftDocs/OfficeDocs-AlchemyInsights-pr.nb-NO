# <a name="recent-data-exposure"></a>Nylige data eksponering

Microsoft ble gjort oppmerksom på en tredje parts forsker som en database som inneholder skjult Microsoft kunde støtte data, var tilgjengelig uten godkjenning. I 31. desember 2019, Microsoft sikret databasen.

Vi er forpliktet til person vernet og sikkerheten til kundene våre og utfører tiltak for å forhindre fremtidige forekomster av dette problemet. Vi har begynt varsler til kunder som har data i denne skjult database. Hvis du vil ha mer informasjon, kan du gå til [informasjon om kunde tilstede status som er relatert til en ny informasjons eksponering](https://aka.ms/privacyinfo).
