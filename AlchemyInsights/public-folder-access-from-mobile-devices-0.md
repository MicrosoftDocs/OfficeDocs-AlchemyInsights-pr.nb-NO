# <a name="public-folder-access-from-mobile-devices"></a>Tilgang til felles mapper fra mobile enheter

Felles mapper er ikke tilgjengelig fra mobile enheter.
  
Hvis du vil ha funksjonalitet som ligner på felles mapper og få tilgang til den fra mobile enheter, kan du se [lære om Microsoft 365-grupper](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) for et bedre alternativ.
