# <a name="cant-read-email-in-google-chrome-with-adblock"></a>Kan ikke lese e-post i Google Chrome med AdBlock

Hvis du ikke kan lese eller skrive e-post når du bruker Google Chrome med AdBlock-filtypen:

- Bruk [Microsoft Edge](https://go.microsoft.com/fwlink/p/?linkid=2001503&amp;clcid=0x409) eller en annen nett leser.
- Utelat Outlook.com fra AdBlock-innstillingene. Velg **AdBlock** i øvre høyre hjørne av Google Chrome, og velg deretter **ikke Kjør på sider på dette domenet**. Oppdater nett leseren for å laste inn Outlook.com på nytt.
