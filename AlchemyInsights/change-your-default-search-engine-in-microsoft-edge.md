# <a name="change-your-default-search-engine-in-microsoft-edge"></a>Endre standard søkemotor i Microsoft Edge

Følg disse instruksjonene for å endre standard søkemotor:
1. Utfør et søk på adresselinjen ved hjelp av søkemotoren du vil angi som standard.
2. Velg **... (Innstillinger og mer) > Innstillinger > Personvern, søk og tjenester**.
3. Rull helt ned til **Tjenester-delen,** og velg deretter **Adresselinje og søk**.
4. Velg **den foretrukne** søkemotoren fra søkemotoren som brukes i adresselinjemenyen.
