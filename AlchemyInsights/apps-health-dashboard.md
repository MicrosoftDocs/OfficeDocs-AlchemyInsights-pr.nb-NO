# <a name="microsoft-365-apps-health-dashboard-in-the-microsoft-365-apps-admin-center"></a>Tilstandsinstrumentbord for Microsoft 365 Apps i administrasjonssenteret for Microsoft 365-apper

Tilstandsinstrumentbordet for Microsoft 365-apper i administrasjonssenteret for Microsoft 365-apper hjelper deg med å holde Office-appene oppdatert ved å gi innsyn i hvordan de fungerer på klientenhetene dine og gi veiledning for å optimalisere og feilsøke problemer. Hvis du vil ha mer informasjon, kan du se [Oversikt over administrasjonssenteret](https://docs.microsoft.com/deployoffice/admincenter/overview) for Microsoft Apps og [tilstand for Microsoft 365-apper.](https://docs.microsoft.com/deployoffice/admincenter/microsoft-365-apps-health)
