# <a name="put-apps-on-the-desktop"></a>Sette apper på skrivebordet

Søk etter appen **** i Start-menyen, og ta tak i appikonet ved å holde nede venstre museknapp og flytte ikonet til skrivebordet.

Du kan også trykke **Windows-tasten + R** og deretter skrive inn denne kommandoen:

`shell:appsfolder`

Dette åpner **programmer-mappen** i **Filutforsker**. Du kan nå dra og slippe appikonet til skrivebordet.
