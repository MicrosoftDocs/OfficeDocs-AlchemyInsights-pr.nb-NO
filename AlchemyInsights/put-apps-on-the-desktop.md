# <a name="put-apps-on-the-desktop"></a>Legge apper på skrive bordet

Søk etter appen i **Start** -menyen, og ta tak i app-ikonet ved å holde nede venstre muse knapp og flytte ikonet til skrive bordet.

Du kan også trykke **Windows-tasten + R** og deretter skrive inn denne kommandoen:

`shell:appsfolder`

Dette åpner **programmer** -mappen i **fil Utforsker**. Du kan nå dra og slippe app-ikonet til skrive bordet.
