# <a name="error-onedrive-cannot-connect-to-windows"></a>Feil: OneDrive kan ikke koble til Windows

Hvis du vil løse problemet, kan du søke etter filen cldflt.sys for å se om det kjører, og hvis ikke, må du aktivere den. 
  
Hvis du vil ha mer informasjon om hvordan du aktiverer cldflt.sys-fil, kan du se [OneDrive kan ikke koble til Windows "feil ved tilgang til filer i OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
