# <a name="error-onedrive-cannot-connect-to-windows"></a>Feil: OneDrive kan ikke koble til Windows

Hvis du vil løse feilmeldingen, kan du søke etter filen cldflt.sys for å se om den kjører, og hvis den ikke er det, aktiverer du den. 
  
Hvis du vil ha mer informasjon om hvordan du aktiverer filen cldflt.sys, kan du se [OneDrive kan ikke koble til Windows"-feil når du åpner filer i OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
