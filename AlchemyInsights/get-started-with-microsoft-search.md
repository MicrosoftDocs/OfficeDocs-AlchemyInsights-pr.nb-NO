# <a name="get-started-with-microsoft-search"></a>Komme i gang med Microsoft Search

Bruk Microsoft Søk til å finne personer, filer, organisasjonskart, nettsteder, svar på vanlige spørsmål og annet innhold som kan hjelpe deg med å fullføre oppgaver i appen du bruker.

Hvis du vil ha mer informasjon, [kan du se Oversikt over Microsoft Search](https://docs.microsoft.com/microsoftsearch/overview-microsoft-search).
