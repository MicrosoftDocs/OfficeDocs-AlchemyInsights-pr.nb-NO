# <a name="error-546-or-5414-related-to-hop-count-exceeded"></a>Feil 5.4.6 eller 5.4.14 relatert til antall mellomstasjoner overskredet

Hvis du får feilkode 5.4.6 eller 5.4.14 relatert til antallet mellomstasjoner overskredet, er det muligens på grunn av et problem med e-postløkke i Exchange Online.

Hvis du vil forstå den mulige årsaken og løsningen, kan du se [Antall mellomstasjoner](/exchange/mail-flow-best-practices/non-delivery-reports-in-exchange-online/fix-error-code-5-4-6-through-5-4-20-in-exchange-online).
