# <a name="deploy-microsoft-365-apps"></a>Distribuere Microsoft 365-apper

For å distribuere Microsoft 365-apper til flere slutt brukere via distribusjons verktøyet for Office, Microsoft Endpoint Configuration Manager, som en del av en disk avbildning av operativ systemet, eller i et delt data maskin scenario ved hjelp av Windows Server med Remote Desktop Services (RDS) eller Terminal Services, bør du vurdere å bruke trinn vise instruksjoner i **Microsoft 365-appene for distribusjon av bedrifts distribusjon**.
  
For å komme i gang må du kontrollere at du er logget på som leier administrator eller global administrator, og klikk deretter her: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
