# <a name="sharepoint-throttling"></a>Regulering av SharePoint

SharePoint Online bruker regulering til å opprettholde optimal ytelse og pålitelighet for SharePoint Online-tjenesten. Regulere begrenser kaller antall brukerhandlinger eller samtidige (som skript eller kode) for å hindre Overforbruk av ressurser.

- [Migrering av data og SPO regulering](https://blogs.technet.microsoft.com/sposupport/2017/08/12/data-migration-and-spo-service-throttling/)
