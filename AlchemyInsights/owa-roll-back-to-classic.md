# <a name="the-new-outlook-on-the-web"></a>Den nye Outlook på nettet

Det er ikke lenger mulig å gå tilbake til den klassiske Outlook-opplevelsen. Hvis du ønsker å gi tilbake melding om den nye opplevelsen, kan du gå til [UserVoice-forumet](https://go.microsoft.com/fwlink/?linkid=2103182). Du kan svare på ideer eller problemer du vil se adressert, og vi svarer direkte på de beste kommentarene og forslagene.

[Få hjelp med den nye Outlook på nettet](https://support.office.com/article/017014cd-2ad0-41ab-8473-6bd8c349d4f8)
