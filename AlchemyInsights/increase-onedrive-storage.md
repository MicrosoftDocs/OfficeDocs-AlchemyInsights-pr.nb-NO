# <a name="increase-onedrive-storage"></a>Øke lagrings plass i OneDrive

Som global-eller SharePoint-administrator i Microsoft 365 kan du angi lagrings plassen i OneDrive for en bestemt bruker direkte fra Microsoft administrasjons senteret.  

Hvis du vil ha mer informasjon, kan du se [endre en bestemt brukers lagrings plass i OneDrive](https://docs.microsoft.com/onedrive/change-user-storage).
