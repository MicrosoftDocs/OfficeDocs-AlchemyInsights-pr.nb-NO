# <a name="block-downloads-for-view-only-files-in-sharepoint-and-onedrive"></a>Blokkere nedlastinger for bare visningsfiler i SharePoint og OneDrive

Når du deler elementer med andre i OneDrive eller SharePoint, kan du gi dem tilgang til å vise, men ikke laste ned. Hvis du vil ha mer informasjon, kan du se Blokkere [nedlastinger for bare visningsfiler i SharePoint og OneDrive.](https://support.microsoft.com/office/block-downloads-for-view-only-files-in-sharepoint-and-onedrive-6051184b-62ac-4149-b874-13dcd40ef91e)
