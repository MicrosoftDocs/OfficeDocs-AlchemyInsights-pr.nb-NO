# <a name="assign-ediscovery-manager-permissions"></a>Tilordne tillatelser for eDiscovery Manager

Hvis du vil tilordne tillatelser for eDiscovery Manager, utfører du følgende trinn:

1. Velg Tillatelser & **> i** sikkerhetssenteret for [Office 365](https://sip.protection.office.com/).
2. Klikk Rediger ved siden av **eDiscovery Manager i ruten eDiscovery** **Manager**. 
3. Klikk **Velg eDiscovery Manager** i redigeringsruten Velg **eDiscovery Manager.**
4. Klikk + Legg til i ruten  **Velg eDiscovery Manager,** og velg deretter brukernes navn.
5. Velg **Legg > fullført > Lagre**.
