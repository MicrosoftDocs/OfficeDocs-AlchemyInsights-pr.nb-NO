# <a name="control-the-access-from-unmanaged-devices"></a>Kontrollere tilgangen fra uadministrerte enheter

Som SharePoint- eller global administrator i Microsoft 365 kan du blokkere eller begrense tilgang til SharePoint- og OneDrive-innhold fra uadministrerte enheter. Hvis du vil ha mer informasjon, kan du se Kontrollere tilgang fra [uadministrerte enheter](https://docs.microsoft.com/sharepoint/control-access-from-unmanaged-devices).
