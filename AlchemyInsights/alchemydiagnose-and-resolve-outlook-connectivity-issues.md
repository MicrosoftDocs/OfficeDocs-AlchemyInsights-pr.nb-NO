# <a name="diagnose-and-resolve-outlook-connectivity-issues"></a>Diagnostisere og løse problemer med Outlook-tilkobling

Hvis du vil diagnostisere og løse problemer med Outlook-tilkoblingen, kan du kjøre [verktøyet Outlook-tilkobling](https://aka.ms/SaRA-OutlookDisconnect-Alchemy) på den berørte data maskinen.
  
Dette verktøyet vil utføre diagnose kontroller og foreslå mulige løsninger for deg for å løse problemet med Outlook-tilkoblingen.
