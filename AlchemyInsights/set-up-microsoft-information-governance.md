# <a name="set-up-microsoft-information-governance"></a>Konfigurere Microsoft informasjons styring

Du finner informasjon om [hvordan](https://admin.microsoft.com/AdminPortal/Home#/modernonboarding/migsetupguide) du konfigurerer og administrerer organisasjonens Styrings strategi for å sikre at dataene klassifiseres og administreres i henhold til spesifikke retnings linjer for livs syklus du angir.

Med denne veiledningen lærer du hvordan du oppretter, bruker automatisk, eller publiserer etiketter, etikett policyer og oppbevarings policyer som brukes i organisasjonens innhold og samsvars oppføringer på nytt. Du får også informasjon om hvordan du importerer CSV-filer med en fil plan for satsvise scenarioer eller for å bruke dem manuelt på enkelt dokumenter.
