# <a name="learn-about-external-calendar-sharing-options"></a>Finn ut mer om alternativer for ekstern kalenderdeling

Hvis du må planlegge møter med personer utenfor organisasjonen, kan du gjøre kalendere tilgjengelige for «eksterne brukere», de som trenger å se ledig/opptatt-tid, men som ikke har brukerkontoer for Microsoft 365-miljøet. Hvis du vil ha mer informasjon, kan du se [Dele kalendere med eksterne brukere](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
