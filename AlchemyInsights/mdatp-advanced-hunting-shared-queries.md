# <a name="sharing-advanced-hunting-queries"></a>Dele avanserte utsøkings spørringer

Avanserte spørrings spørringer kan deles mellom brukere i samme organisasjon. Du kan også finne spørringer som deles offentlig på GitHub. Med disse spørringene kan du raskt gjøre om bestemte trusler for å lete uten å måtte skrive spørringer fra grunnen av.
  
Hvis du vil ha mer informasjon, kan du se [bruke delte spørringer i avansert jakt](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/advanced-hunting-shared-queries).
