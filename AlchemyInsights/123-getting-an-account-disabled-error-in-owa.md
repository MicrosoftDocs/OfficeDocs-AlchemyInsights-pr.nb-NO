# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Opprette en konto deaktivert feil i Outlook på weben?

Hvis du får feilen **har blitt deaktivert kontoen din** med `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` når du prøver å åpne Outlook på weben (tidligere kjent som Outlook Web App), administratoren din organisasjon kan ha deaktivert tilgangen til Outlook på weben. Administrator kan løse denne feilen ved å følge fremgangsmåten i emnet [aktivere eller deaktivere Outlook Web App for en postboks](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
