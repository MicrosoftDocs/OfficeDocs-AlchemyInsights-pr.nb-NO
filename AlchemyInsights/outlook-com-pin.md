# <a name="change-your-pin"></a>Endre PIN-koden

Outlook.com bruke ikke en PIN-kode, men du kan logge deg på din PC med en PIN-kode eller Windows Hallo. Slik endrer du PIN-koden i Windows 10:

1. Velg Start-knappen eller trykke Windows-tasten på tastaturet, og velg deretter **Innstillinger for** > **kontoer** > **Alternativer for pålogging**.
2. Velg **Endre** -knappen under **PIN-kode**.
3. Angi den gjeldende PIN-koden, og deretter angi og Bekreft den nye PIN-koden under.

[Les mer om Windows Hallo.](https://support.microsoft.com/help/17215/)
