# <a name="change-the-font-size-in-a-message"></a>Endre skrift størrelsen i en melding

1. Hvis du allerede har skrevet inn meldingen, merker du teksten.
2. Velg  **Format** -ikonet under meldingen (det er det første ikonet).
3. Velg  **Skriftstørrelse**  -ikonet ovenfor meldingen (det er det sjuende ikonet).
4. Velg en størrelse.
