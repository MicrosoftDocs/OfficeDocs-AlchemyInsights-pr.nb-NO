# <a name="error-message-when-sharing-files-from-windows-explorer"></a>Feilmelding når du deler filer fra Windows Utforsker

Når du prøver å dele fra Windows Utforsker, kan du få følgende feilmelding:
  
"Beklager, vi kan ikke nå serveren akkurat nå. Vennligst forsøk på nytt senere"
  
Hvis du vil løse feilmeldingen, går du **Advanced** til Alternativer for Verktøy **alternativer for Internett** \> for **Internet,** \> og deretter fjerner du merket for **Ikke lagre krypterte sider på disken**.
  
 **Merk:** Du må kanskje starte nettleseren på nytt for at innstillingen skal tre i kraft. Hvis du vil ha mer informasjon, kan du se [feilmeldingen "Kan ikke nå serveren akkurat nå" når du deler filer](https://go.microsoft.com/fwlink/?linkid=2022914).
