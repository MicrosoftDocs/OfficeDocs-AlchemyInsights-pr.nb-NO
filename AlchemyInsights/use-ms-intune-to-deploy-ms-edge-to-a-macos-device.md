# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Bruke Microsoft Intune til å distribuere Microsoft Edge til en macOS-enhet

Hvis du vil distribuere, konfigurere, overvåke og beskytte en app på en macOS-enhet, må du legge til appen i Microsoft Intune. Appen må være av en type som støttes av Intune, og administrasjons alternativene som er tilgjengelige i Intune, avhenger av typen til appen. Hvis du bruker Intune, betyr det også at du ikke trenger å bruke verktøyet macOS app-wrapper.

Intune støtter Microsoft Edge. Hvis du vil vite mer, kan du se [legge til Microsoft Edge i MacOS-enheter ved hjelp av Microsoft Intune](https://go.microsoft.com/fwlink/?linkid=2134949).
