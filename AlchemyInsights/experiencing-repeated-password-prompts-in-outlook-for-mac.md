# <a name="experiencing-repeated-password-prompts-in-outlook-for-mac"></a>Gjentatt har spørsmål om passord i Outlook for Mac?

Vi kan bidra til å gi mer spesialisert Outlook for Mac hjelp direkte fra programmet.
  
Gå til den aktuelle brukeren Mac, åpner du Outlook, og klikk på **Hjelp** \> **Kontakt kundestøtte**.
  
Hvis du vil vite mer om hvordan du bruker Outlook for Mac-app kundestøtte, kan du lese denne artikkelen: [Ny kontakt støtte-funksjonen i Outlook-2016 for Mac](https://answers.microsoft.com/msoffice/forum/msoffice_outlook-mso_mac-mso_mac2016/new-contact-support-feature-in-outlook-2016-for/d4fc21c4-25e2-4e10-b943-1fba6542b517.aspx)
