# <a name="getting-an-account-disabled-error-in-outlook-on-the-web"></a>Få en konto deaktivert feil i Outlook på nettet?

Hvis du får feilmeldingen **Kontoen er** `X-OWA-Error: Microsoft.Exchange.Data.Storage.AccountDisabledException` deaktivert når du prøver å åpne Outlook på nettet (tidligere kjent som Outlook Web App), kan det hende at organisasjonsadministratoren har deaktivert tilgangen til Outlook på nettet. Administratoren kan løse denne feilen ved å følge trinnene i emnet [Aktivere eller deaktivere Outlook Web App for en postboks](https://technet.microsoft.com/library/bb124124%28v=exchg.150%29.aspx).
