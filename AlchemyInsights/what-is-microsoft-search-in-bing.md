# <a name="what-is-microsoft-search-in-bing"></a>Hva er Microsoft Søk i Bing?

[Microsoft Søk i Bing er](https://docs.microsoft.com/deployoffice/microsoft-search-bing#what-is-microsoft-search-in-bing) en funksjon som hjelper deg med å søke på tvers av organisasjonen og på nettet. Bruk Bings adresselinje og søkeboks til å finne dokumenter, interne og eksterne nettsteder, forretningsverktøy, personer og organisasjonskart, filer, SharePoint-nettsteder, OneDrive-innhold, Teams- og Yammer-samtaler og mange andre ressurser.
