# <a name="remove-an-extension-from-microsoft-edge"></a>Fjerne en utvidelse fra Microsoft Edge

Slik fjerner du en utvidelse fra Microsoft Edge:

1. Åpne Microsoft Edge.
2. Velg **... > utvidelser**.
3. Finn filtypen du ønsker å fjerne, på siden **installerte utvidelser** , og velg deretter **Fjern** ved siden av den.
4. Velg **Fjern** i bekreftelses dialog boksen.
