# <a name="learn-about-external-calendar-sharing-options"></a>Lær mer om alternativer for eksterne kalenderdeling

Hvis du skal planlegge møter med personer utenfor organisasjonen, Office 365, kan du gjøre kalendere som er tilgjengelige for "eksterne brukere" de som trenger å se ledige og opptatte tidspunkt, men har ikke brukerkontoer for Office 365-miljø. Hvis du vil ha mer informasjon, se [dele kalendere med eksterne brukere](https://support.office.com/article/fb00dd4e-2d5f-4e8d-8ff4-94b2cf002bdd.aspx).
