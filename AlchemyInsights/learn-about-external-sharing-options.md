# <a name="learn-about-external-calendar-sharing-options"></a>Lær om alternativer for deling av eksterne kalendere

Hvis du må planlegge møter med personer utenfor organisasjonen, kan du bruke Office 365 til å gjøre kalendere tilgjengelige for eksterne brukere, de som trenger å se ledige og opptatte tidspunkt, men som ikke har brukerkontoer for Office 365-miljøet. Hvis du vil ha mer informasjon, kan du se [dele kalendere med eksterne brukere](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
