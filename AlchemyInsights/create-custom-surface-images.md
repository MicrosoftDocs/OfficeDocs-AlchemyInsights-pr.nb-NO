# <a name="create-custom-microsoft-surface-operating-system-images"></a>Opprette egendefinerte Microsoft Surface-operativsystembilder

Bruk Microsoft Surface Deployment Accelerator (SDA) til å forenkle og automatisere distribusjonen av et nytt operativsystem (OS) som er skreddersydd for organisasjonen din og oppfyller Microsofts anbefalinger.

1. Gå til SurfaceDeploymentAccelerator i GitHub.

1. Velg **Kode**, og velg deretter **Åpne med GitHub Desktop** eller Last ned **ZIP.**

1. Se gjennom **README.md** filen for å lære hvordan skriptet fungerer.

1. Rediger skriptet med de riktige variablene for miljøet som dokumentert i Viktig-filen, og se deretter gjennom det før du kjører det i testmiljøet.
