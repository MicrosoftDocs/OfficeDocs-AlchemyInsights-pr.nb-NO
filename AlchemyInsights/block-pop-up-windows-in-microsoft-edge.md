# <a name="block-pop-up-windows-in-microsoft-edge"></a>Blokkere popup-vinduer i Microsoft Edge

1. Gå til **... (Innstillinger og mer)**  >  **Innstillinger**  >  **Områdetillatelser**.
2. Velg **Popup-vinduer og omdirigeringer**.
3. Aktiver **Blokker**.
