# <a name="troubleshoot-administrator-issues"></a>Feilsøke administratorproblemer

Utfør følgende fremgangsmåte for å feilsøke administratorrelaterte problemer:

1. Hvis du vil se globale administratorer i katalogen, kan du gå til **siden Roller og administratorer.**
2. På siden **Roller og administratorer** finner du rollen Global **administrator** i listen over roller. Åpne global **administrator-siden** for å se alle de globale administratorene i katalogen.
3. [Vis administratorroller i Azure AD:](https://docs.microsoft.com/azure/active-directory/roles/manage-roles-portal)Du kan nå se og administrere alle medlemmene av **administratorrollen** i Azure AD-portalen.
4. Forstå forskjellen mellom Azure AD-roller og [Azure RBAC-roller:](https://docs.microsoft.com/azure/role-based-access-control/rbac-and-directory-admin-roles)Hvis du er ny med Azure, kan det være litt utfordrende å forstå de ulike rollene i Azure. Denne artikkelen forklarer de ulike rollene og når du bør bruke hver av disse rollene.
5. [Opprett en ny Azure Active Directory:](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-access-create-new-tenant)I denne hurtigstarten lærer du hvordan du kommer til Azure Portal og Azure AD, og du lærer hvordan du oppretter en grunnleggende leier for organisasjonen.
6. [Legge til eller endre Azure-abonnementsadministratorer:](https://docs.microsoft.com/azure/cost-management-billing/manage/add-change-subscription-administrator)Denne artikkelen beskriver hvordan du legger til eller endrer administratorrollen for en bruker som bruker Azure RBAC i abonnementsomfanget.
