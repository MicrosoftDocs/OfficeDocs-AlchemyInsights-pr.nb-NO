# <a name="pop-and-imap4-settings"></a>POP- og IMAP4-innstillinger

Hvis du vil finne serverinnstillingene for POP3 og IMAP4 for din egen postboks, kan du se [INNSTILLINGER FOR POP- og IMAP-e-post for Outlook](https://support.office.com/article/8361e398-8af4-4e97-b147-6c6c4ac95353.aspx).

Administratorer kan aktivere og konfigurere POP3 og IMAP4 i lokale Exchange-organisasjoner. Hvis du vil ha mer informasjon, kan du se [Aktivere og konfigurere POP3 på Exchange-servere](https://technet.microsoft.com/library/bb124934.aspx) og [Aktivere og konfigurere IMAP4 på Exchange-servere](https://technet.microsoft.com/library/bb124489.aspx).
