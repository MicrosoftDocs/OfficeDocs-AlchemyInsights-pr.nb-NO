# <a name="learn-how-microsoft-search-in-bing-helps-keep-your-information-secure"></a>Finn ut hvordan Microsoft Search i Bing bidrar til å holde informasjonen sikker

Med Microsoft Search i Bing:

- Bare påloggede brukere kan **se søkeresultater.**
- Arbeidsidentiteten din eller organisasjonens identitet brukes ikke av Microsoft til å målrette reklame.
- Søkeadministratoren kan bare se antall søk etter type, for eksempel etter person eller filtype, og en aggregert liste over toppsøk.
- Bare du kan se søkeloggen **for arbeidet.**
- Du finner bare arbeidsfilene og informasjonen som er delt med deg, eller som du har tillatelse til å se.
- Informasjonen blir aldri tilgjengelig for andre organisasjoner, inkludert Microsoft, eller for offentlige søk utført på Bing.

Hvis du vil ha mer informasjon, kan du se [Hvordan Microsoft Søk i Bing bidrar til å holde informasjonen sikker.](https://support.microsoft.com/office/how-microsoft-search-in-bing-helps-keep-your-info-secure-cbce46ae-bb1f-4d0e-86f1-5984f4589113)
