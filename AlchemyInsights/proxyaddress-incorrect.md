# <a name="proxyaddress-incorrect"></a>ProxyAddress feil

Når et objekt er synkronisert til Azure AD, verdiene som er angitt i attributtet physicalDeliveryOfficeName i Active Directory sammenlignes med Azure AD-regler, og deretter physicalDeliveryOfficeName-attributtet er fylt ut i Azure AD. Verdiene for attributtet physicalDeliveryOfficeName for objektet i Active Directory kan derfor ikke være den samme som verdiene for attributtet physicalDeliveryOfficeName i Azure AD.
  
Hvis du vil vite mer om hvordan proxyAddress fylles ut, kan du se [hvordan proxyAddress-attributtet er fylt ut i Azure ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
