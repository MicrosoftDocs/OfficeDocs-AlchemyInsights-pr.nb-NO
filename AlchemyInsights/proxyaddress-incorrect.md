# <a name="proxyaddress-incorrect"></a>ProxyAddress feil

Når et objekt synkroniseres med Azure AD, sammenlignes verdiene som er angitt i attributtet proxyAddresses i Active Directory, med Azure AD-regler, og deretter fylles proxyAddresses-attributtet ut i Azure AD. Verdiene for proxyAddresses-attributtet for objektet i Active Directory kan derfor ikke være de samme som verdiene for proxyAddresses -attributtet i Azure AD.
  
Hvis du vil vite mer om hvordan proxy-adressen fylles ut, kan du se [Hvordan proxyAddress-attributtet fylles ut i Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
