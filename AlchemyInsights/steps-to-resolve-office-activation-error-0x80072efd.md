# <a name="steps-to-resolve-office-activation-error-0x80072efd"></a>Fremgangs måte for å løse feil i Office-aktivering 0x80072EFD

Prøv disse løsningene for å løse Office-aktiverings feilen:
  
1. Kontroller [brann muren](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckFirewall).

2. Kontroller [antivirus program varen](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckAV).

3. Kontroller [proxy-innstillingene](https://support.office.com/article/0d23d3c0-c19c-4b2f-9845-5344fedc4380#BKMK_CheckProxy)

4. Last ned og Installer [assistenten for støtte og gjenoppretting](https://aka.ms/SARA-OfficeActivation-Alchemy) , og Kjør **feil søkings verktøyet for aktivering**.
