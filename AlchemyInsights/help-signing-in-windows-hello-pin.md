# <a name="help-signing-in-with-windows-hello-pin"></a>Hjelp til å logge på med PIN-koden for Windows Hello

- Hvis du vil endre, tilbakestille eller gjenopprette PIN-koden (når du er logget på), kan du gå til Endre [PIN-koden](https://support.microsoft.com/windows/change-your-pin-when-you-re-already-signed-in-to-your-device-0bd2ab85-b0df-c775-7aef-1324f2114b19)når du allerede er logget på enheten.

- Hvis du vil endre, tilbakestille eller gjenopprette PIN-koden (når du er logget av), kan du gå til Tilbakestille PIN-koden når du ikke er logget [på Windows 10.](https://support.microsoft.com/windows/reset-your-pin-when-you-aren-t-signed-in-to-windows-10-a386c519-3ab2-b873-1e9b-bb228a98b904)

- Hvis du vil konfigurere PIN-koden for Windows Hello, kan [du gå til Lær om Windows Hello og konfigurere den.](https://support.microsoft.com/windows/learn-about-windows-hello-and-set-it-up-dae28983-8242-bb2a-d3d1-87c9d265a5f0)
