"Operativsystem som ikke støttes"

# <a name="solutions-for-unsupported-operating-system"></a>Løsninger for "Operativsystem som ikke støttes"

Når du prøver å installere Office 365/2016, kan du se følgende feilmelding hvis enheten ikke oppfyller [systemkravene](https://products.office.com/office-system-requirements).
  
 **"Operativsystem som ikke støttes"**
  
Office 365/2016 krever:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012, eller Windows Server 2008 R2
  
ELLER
  
Mac OS X 10,10
  
Hvis du fikk denne meldingen når du prøver å installere Office på en mobil enhet eller en Chromebook, kan du se [konfigurere Office-programmer og e-post på en mobil enhet](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
