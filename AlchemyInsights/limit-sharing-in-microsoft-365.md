# <a name="limit-sharing-in-microsoft-365"></a>Begrense deling i Microsoft 365

Selv om du ikke kan deaktivere intern deling helt eller fjerne Del-knappen fra områder, finnes det en rekke måter du kan begrense deling i Microsoft 365 for å møte behovene til organisasjonen. Hvis du vil ha mer informasjon, kan du se [Begrense deling i Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
