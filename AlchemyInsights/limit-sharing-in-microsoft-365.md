# <a name="limit-sharing-in-microsoft-365"></a>Begrense deling i Microsoft 365

Selv om du ikke kan deaktivere intern deling helt eller fjerne del-knappen fra nett steder, finnes det en rekke måter du kan begrense deling i Microsoft 365 på for å møte organisasjonens behov. Hvis du vil ha mer informasjon, kan du se [begrense deling i Microsoft 365](https://docs.microsoft.com/Office365/Enterprise/microsoft-365-limit-sharing).
