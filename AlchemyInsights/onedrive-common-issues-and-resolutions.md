# <a name="onedrive-common-issues-and-resolutions"></a>Vanlige problemer og løsninger for OneDrive

Her er noen ofte forespurte emner om OneDrive-problemer:

- **Konfigurere OneDrive-synkronisering:** [Synkroniser OneDrive for Business-filer](https://go.microsoft.com/fwlink/?linkid=533375) eller [synkroniser SharePoint-filer](https://go.microsoft.com/fwlink/?linkid=871666) med Synkroniseringsklienten for OneDrive for Windows.  Bruker du en Mac? Du kan [også synkronisere filer med OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) med den.

- **Feilsøke Synkronisering av OneDrive**: Hvis OneDrive ikke synkroniserer filer slik du forventer, kan du [feilsøke synkroniseringsproblemer](https://go.microsoft.com/fwlink/?linkid=866431) med OneDrive for Windows. Bruker du en Mac? Du kan [også feilsøke synkroniseringsproblemer](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756) for det.
- **OneDrive Limits**: Lær om [begrensningene og begrensningene](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) som gjelder for filnavn og filtyper når du synkroniserer med OneDrive.
- **Lagringsplass på OneDrive**: Se hvordan [du endrer en bestemt brukers Lagringsplass for OneDrive](https://docs.microsoft.com/onedrive/change-user-storage) eller [administrerer OneDrive for Business-lagring](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Overvåkingslogger**: Se hvordan du kan [bruke overvåkingsloggsøk](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) til å kontrollere aktiviteten til filen(e), mappen(e), brukeren(e) eller tillatelsene.
