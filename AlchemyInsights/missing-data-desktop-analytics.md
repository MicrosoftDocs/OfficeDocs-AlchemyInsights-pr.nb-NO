# <a name="cant-create-a-compatibility-assessment"></a>Kan ikke opprette en Kompatibilitetskontroll

Microsoft Endpoint Configuration Manager konfigurerte enheten, men skrive bords analyse kan ikke opprette en kompatibilitetskontroll. Den har ikke en fullstendig data sett for enhetens konfigurasjon (statistikk) eller installerte apper (lager).

Dette problemet blir ofte løst automatisk når enheten prøver på nytt. Hvis det vedvarer, må du kontrollere at enheten kan kommunisere med tjenesten. Hvis du vil ha mer informasjon, kan du se [ende punkter](https://docs.microsoft.com/configmgr/desktop-analytics/enable-data-sharing#endpoints) og [manglende data](https://docs.microsoft.com/configmgr/desktop-analytics/monitor-connection-health#missing-data).
