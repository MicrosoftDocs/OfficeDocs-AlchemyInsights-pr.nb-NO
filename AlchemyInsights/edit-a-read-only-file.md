# <a name="edit-a-read-only-file"></a>Redigere en skrivebeskyttet fil

I dokumentbiblioteker, når du klikker på et dokumentnavn, kan Office-programmet uventet åpne den som en skrivebeskyttet fil. Hvis du vil åpne den for redigering, klikker du **Åpne** (de tre prikkene), og klikk deretter **Rediger**
