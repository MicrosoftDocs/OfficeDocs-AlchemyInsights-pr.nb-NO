# <a name="edit-a-read-only-file"></a>Redigere en skrivebeskyttet fil

Når du klikker navnet på et dokument i dokumentbiblioteker, kan Office-programmet uventet åpne det som en skrivebeskyttet fil. Hvis du vil åpne den for redigering, klikker du **Åpne** (de tre prikkene), og deretter klikker du **Rediger.**
