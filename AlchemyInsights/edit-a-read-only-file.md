# <a name="edit-a-read-only-file"></a>Redigere en skrivebeskyttet fil

Når du klikker navnet på et dokument i dokument biblioteker, kan det hende at Office-programmet uventet åpner det som en skrivebeskyttet fil. Hvis du vil åpne den for redigering, klikker du **Åpne** (de tre prikkene), og deretter klikker du **Rediger.**
