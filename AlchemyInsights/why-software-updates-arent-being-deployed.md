# <a name="why-software-updates-are-not-being-deployed"></a>Hvorfor programvare oppdateringer distribueres ikke?

Se gjennom følgende hvis du har konfigurert programvare oppdateringer via en Windows 10 Update-ring, men oppdateringene distribueres ikke:  

- Vurder å endre Windows-vedlikehold fra en  **halv Årst kanal**  Utgivelses type til en strengere, mer oftere frigivelses type  
- Se utsettelses-perioden for  **kvalitets oppdatering**  og  **funksjons oppdatering**. Utsettelses perioden kan føre til forsinkelser i oppdateringer for opptil 180 dager.
