# <a name="change-your-profile-name-in-outlookcom"></a>Endre profilnavnet ditt i Outlook.com

1. Gå til [informasjonen om Microsoft-kontoen din](https://go.microsoft.com/fwlink/p/?linkid=860841).
2. Velg **Rediger navn**.
3. Skriv inn fornavn og etter navn, og velg **Lagre**.

Det kan ta opptil 24 timer for endringer i profilnavnet for å gå i kraft.
