# <a name="activating-windows-10-subscriptions"></a>Aktivere Windows 10-abonnementer

Microsoft 365 Business, Microsoft 365 Education, Microsoft 365 og Enterprise tilbyr digitale lisenser for Windows 10 som en del av abonnementet. For å aktivere abonnementet må enheten din allerede ha aktivert Windows 10 Pro (1703 eller nyere). Hvis du vil se en fullstendig liste over kravene for aktivering av et abonnement på Windows 10 se [aktiverings krav for Windows 10-abonnement](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
