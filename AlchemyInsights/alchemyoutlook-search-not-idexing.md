# <a name="outlook-search-not-indexing"></a>Outlook-søk indekserer ikke

Vi er klar over problemer med Outlook-søk som er relatert til nylige sikkerhetsoppdateringer for Windows.
  
Når du søker i Outlook, kan du få en av disse feilene: **noe gikk galt og søket kunne ikke fullføres**, eller **søkeresultatene kan være ufullstendige fordi elementer fortsatt indekseres**. Reparasjonen ble utstedt som en valgfri Windows-oppdatering. Vennligst installer den som tilsvarer den versjonen av Windows du ser dette problemet på: 
  
> [Windows 7-27 juni 2017 oppdatering](https://support.microsoft.com/kb/4022168.aspx)
    
> [Windows 8,1-27 juni, 2017 oppdatering](https://support.microsoft.com/kb/4022720.aspx)
    
> [Windows 10 versjon 1703-27 juni 2017 oppdatering](https://support.microsoft.com/kb/4022716.aspx)
    
For den nyeste informasjonen, kan du se artikkelen: [kjente problemer med Outlook i juni 2017 sikkerhetsoppdateringer](https://support.office.com/article/Outlook-known-issues-in-the-June-2017-security-updates-3F6DBFFD-8505-492D-B19F-B3B89369ED9B.aspx) problemet #5.
