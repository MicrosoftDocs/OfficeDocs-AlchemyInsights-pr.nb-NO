# <a name="talk-to-family-and-friends-on-skype-in-outlookcom"></a>Snakke med familie og venner på Skype i Outlook.com

Outlook.com-kunder kan snakke med venner og familie med chat, stemme- og videofunksjonene i Skype. Når du [fletter Microsoft og Skype-kontoer](https://go.microsoft.com/fwlink/p/?linkid=2001101&amp;clcid=0x409), automatisk har du tilgang til Skype World Wide web i Outlook.com.
  
Lær mer om hvordan du [snakker til familie og venner på Skype i Outlook.com](https://go.microsoft.com/fwlink/p/?linkid=2001407&amp;clcid=0x409).
