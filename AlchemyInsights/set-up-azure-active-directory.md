# <a name="set-up-azure-active-directory"></a>Konfigurere Azure Active Directory

[Installasjons veiledningen for Azure ad](https://go.microsoft.com/fwlink/?linkid=2134390) inneholder informasjon for å sikre at organisasjonen har et sterk sikkerhets grunnlag. I denne veiledningen kan du konfigurere innledende funksjoner, for eksempel Azure-basert tilgangs kontroll (Azure RBAC) for administratorer, Azure AD Connect for den lokale katalogen og Azure AD Connect-tilstand, slik at du kan overvåke din hybrid identitets tilstand under automatiserte synkroniseringer.

Den inneholder også viktig informasjon om hvordan du aktiverer selv betjent tilbakestilling av passord, betinget tilgang og integrert tredje parts pålogging, inkludert valg fri avansert identitets beskyttelse og automatisering av bruker klar gjøring.
