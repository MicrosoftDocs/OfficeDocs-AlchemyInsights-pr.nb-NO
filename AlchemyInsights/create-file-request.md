# <a name="how-to-create-a-file-request"></a>Slik oppretter du en fil forespørsel

Med funksjonen for fil forespørsel i OneDrive kan du velge en mappe der andre kan laste opp filer ved hjelp av en kobling du sender dem. Brukerne du ber om filer fra, kan ikke se hva som er i mappen – de kan bare laste opp filer til den.

**Obs**! en administrator må aktivere [alle koblinger](https://docs.microsoft.com/sharepoint/turn-external-sharing-on-or-off) i OneDrive for fil forespørsler for å fungere.

- Hvis du vil ha mer informasjon, kan du se [opprette en fil forespørsel](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af).
