# <a name="set-up-exchange-online-protection"></a>Konfigurere Exchange Online Protection

Microsoft Exchange Online Protection (EOP) er en sky BAS ert e-postfiltrerings tjeneste for beskyttelse mot søppel post og skadelig program vare, med funksjoner for å beskytte organisasjonen mot brudd på meldings policy.

Bruk [veiledningen for konfigurasjon av Exchange Online](https://admin.microsoft.com/adminportal/home#/modernonboarding/prepareyourenvironment) til å konfigurere EoP ved å velge mellom de tre implementerings scenariene – lokale post bokser, hybride (blanding av lokale og Sky) post bokser eller alle Sky post bokser, som passer i organisasjonen. Veiledningen gir informasjon og ressurser for å konfigurere EOP og se gjennom brukerens lisensiering, tilordne tillatelser i administrasjons senteret for Microsoft 365 og konfigurere organisasjonens retnings linjer for beskyttelse mot skadelig program vare og søppel post i sikkerhets & Samsvars senteret.
