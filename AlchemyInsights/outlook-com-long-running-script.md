# <a name="long-running-script"></a>Skript som kjører lenge

Vi er klar over problemer med "lenge kjørende skript" i Outlook.com. Du kan unngå dette ved å bruke beta versjonen av Outlook.com.
  
[Lær mer om beta versjonen av Outlook.com.](https://support.office.com/article/40676ad0-c831-45ac-a023-5be633be798d?wt.mc_id=Office_Outlook_com_Alchemy)
