# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>Bruk Microsoft Intune til å konfigurere policyinnstillinger for Microsoft Edge for Windows 10

Du kan konfigurere Microsoft Edge-policyer og -innstillinger ved å legge til en enhetskonfigurasjonsprofil i Microsoft Intune. Dette tilsvarer konfigurering av gruppepolicyer for Microsoft Azure Active Directory eller lokale gruppepolicyobjektinnstillinger (GPO) på brukerenheter.

Hvis du vil ha mer informasjon, kan du se [Opprette en profil for å administrere innstillinger i Microsoft Edge for Windows 10](https://go.microsoft.com/fwlink/?linkid=2133700).

Denne artikkelen gjelder for Microsoft Edge versjon 77 og nyere.
