# <a name="site-is-not-available"></a>Nettstedet er ikke tilgjengelig

Hvis SharePoint- eller OneDrive-området ikke er tilgjengelig for flere brukere, kan det være et midlertidig tjenesteproblem. Kontroller [instrumentbordet for tjenestetilstand](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) for å se om leieren din påvirkes.
