# <a name="site-is-not-available"></a>Området er ikke tilgjengelig

Hvis OneDrive eller -området ikke er tilgjengelig for flere brukere, kan det være et problem med tjenesten midlertidig. Kontroller[Service helse instrumentbord](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) for å se Hvis din leier berøres.
