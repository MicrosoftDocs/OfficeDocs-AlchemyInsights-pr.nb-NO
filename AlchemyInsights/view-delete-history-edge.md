# <a name="view-and-delete-browsing-history-in-microsoft-edge"></a>Vise og slette leserloggen i Microsoft Edge

Med din tillatelse husker den nye Microsoft Edge nettleserdataene dine, noe som hjelper deg med å gå tilbake til et favorittnettsted eller fylle ut et skjema. I InPrivate, Gjest og andre moduser lagrer Microsoft Edge mindre data enn de lagres under vanlig surfing.

Nettlesingsdataene lagres på enheten. Hvis du har aktivert synkronisering, lagres dataene i skyen og synkroniseres på tvers av enhetene der du er logget på Microsoft Edge-kontoen.

Hvis du vil se leserloggen, **velger du Innstillinger og mer ...**   >  **Logg**  >  **Behandle loggen**. Du kan når som helst tømme leserloggen.
