# <a name="cant-add-default-2010-approval-workflow"></a>Kan ikke legge til standard godkjenningsarbeidsflyt i 2010

Du kan legge et globalt gjenbrukbar arbeidsflyt (for eksempel "godkjenning - SharePoint 2010") til lister eller biblioteker i en områdesamling for Microsoft SharePoint. Arbeidsflyten for dokumentgodkjenning kan mangle. Hvis du vil løse dette problemet, kan du referere til [kan ikke legge til et globalt gjenbrukbar arbeidsflyt på SharePoint-lister eller -biblioteker](https://support.microsoft.com/help/4467263/sharepoint-designer-2013-shows-empty-wfpub-library).

For mer informasjon om godkjenninger av arbeidsflyt, kan du se [Alt om arbeidsflyter for godkjenning](https://support.office.com/article/All-about-Approval-workflows-078C5A89-821F-44A9-9530-40BB34F9F742). 
 
Lær mer om [Moderne godkjenninger i arbeidsflyt](https://flow.microsoft.com/blog/introducing-modern-approvals).
