# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Endre innstillinger for skjerm språk i SharePoint eller OneDrive 

Du kan ikke endre standard språk innstillingene for SharePoint eller OneDrive fordi disse innstillingene ble definert da du opprettet Microsoft 365-abonnementet. Du kan imidlertid endre innstillingene for skjerm språk.

Hvis du vil endre de personlige innstillingene dine, kan du se  [endre innstillingene for personlig språk og område](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Hvis du vil endre områdets bruker grensesnitt, kan [du se velge språkene du vil gjøre tilgjengelige for et bruker grensesnitt for området](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Hvis du vil ha mer informasjon om feil søking av språk innstillinger, [kan du se ikke endre standard språket for en SharePoint Online](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection)-områdesamling.
