# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Endre innstillinger for visningsspråk i SharePoint eller OneDrive 

Du kan ikke endre standard språkinnstillinger for SharePoint eller OneDrive fordi disse innstillingene ble definert da du opprettet Microsoft 365-abonnementet. Du kan imidlertid endre innstillingene for visningsspråk.

Hvis du vil endre personlige innstillinger, kan du se [Endre innstillingene for personlige språk og område](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Hvis du vil endre nettstedets brukergrensesnitt, kan du se [Velge språkene du vil gjøre tilgjengelig for brukergrensesnittet for et område](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Hvis du vil ha mer informasjon om feilsøking av språkinnstillinger, kan du se [Du kan ikke endre standardspråket for en områdesamling for SharePoint Online](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).
