# <a name="change-display-language-settings-in-sharepoint-or-onedrive"></a>Endre innstillinger for skjermspråk i SharePoint eller OneDrive 

Du kan ikke endre standard Språkinnstillinger for SharePoint eller OneDrive fordi disse innstillingene ble definert da du opprettet Office 365-abonnementet. Du kan imidlertid endre innstillingene for skjermspråk.

For å endre dine personlige innstillinger, se [endre dine personlige innstillinger for språk og region](https://support.office.com/article/Change-your-personal-language-and-region-settings-caa1fccc-bcdb-42f3-9e5b-45957647ffd7).

Hvis du vil endre områdets brukergrensesnitt, kan [du se velge språkene du vil gjøre tilgjengelige for brukergrensesnittet for et område](https://support.office.com/article/choose-the-languages-you-want-to-make-available-for-a-site-s-user-interface-16d3a83c-05ab-4b50-8fbb-ff576a3351e8).

Hvis du vil ha mer informasjon om feilsøking av språkinnstillinger, se [du kan ikke endre standardspråket for en SharePoint Online områdesamling](https://docs.microsoft.com/sharepoint/support/sites/change-default-language-site-collection).
