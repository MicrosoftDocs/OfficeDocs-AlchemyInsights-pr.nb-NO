# <a name="when-do-my-profile-changes-sync-to-the-sharepoint-user-profile-application"></a>Når blir profilen min synkronisert til bruker profil programmet for SharePoint?

Synkronisering av profil bilde i SharePoint Online krever en brukers handling for å starte synkroniseringen av bildet deres. Det finnes ingen automatisk synkronisering av bilder til SharePoint Online. SharePoint Online synkroniserer bare bilder fra Exchange Online, ikke Azure Active Directory.

Hvis en bruker aldri har utført en handling som ber om en synkronisering, eller de har oppdatert profil bildet sitt, og det nye dokumentet ikke vises i SharePoint Online, kan det hende at brukeren må be om å bli synkronisert. Slik kan de gjøre dette:

1. Åpne en nett leser i en privat eller inkognito-økt, og [gå til Office.com](https://www.office.com/).

2. Logg på med jobb-eller skole kontoen, og klikk på OneDrive-eller SharePoint-flisen.

3. [Legg til profil bildet sitt i Microsoft 365](https://support.office.com/article/Add-your-profile-photo-to-Office-365-2eaf93fd-b3f1-43b9-9cdc-bdcd548435b7).

[Informasjon om profil bilde synkronisering i Microsoft 365](https://support.office.com/article/Information-about-user-profile-synchronization-in-SharePoint-Online-177eb196-5887-43c9-84c3-b98a43d35129)
