"Kan ikke starte Microsoft Outlook"

# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Outlook-feil: "Kan ikke starte Microsoft Outlook"

Du har problemer med å starte Outlook-2016, Outlook 2013 eller Outlook 2010, eller motta "kan ikke starte Microsoft Outlook. Kan ikke åpne Outlook-vinduet"feilmeldinger?
  
Bruk først feilsøkingstrinnene i [denne artikkelen](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) for å løse disse problemene for oppstart. 
  
Du kan også kjøre [diagnoseverktøyet for avansert](https://aka.ms/SaRA-OutlookAdvDiagnostics) på den berørte datamaskinen.
