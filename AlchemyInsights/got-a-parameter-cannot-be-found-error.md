# <a name="migrating-email-from-imap-email-system-to-microsoft-365"></a>Overføre e-post fra EMAP-e-postsystem til Microsoft 365

Hvis du vil overføre e-post fra et IMAP-e-postsystem, følger du denne prosessen: [IMAP-overføring i administrasjonssenteret for Microsoft 365](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/imap-migration-in-the-admin-center)
  
Mange store organisasjoner må planlegge nøye for å håndtere brukerens forventninger. [Artikkelen Hva du trenger å vite om overføring av IMAP-postbokser til Microsoft 365-artikkelen,](https://docs.microsoft.com/Exchange/mailbox-migration/migrating-imap-mailboxes/migrating-imap-mailboxes) hjelper deg med å administrere endringen.

Se også [siden Adopt Microsoft 365 med FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) for assistert hjelp med overføring, hvis du er kvalifisert.
