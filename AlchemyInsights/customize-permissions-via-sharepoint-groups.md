# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Slik tilpasser du tillatelser for SharePoint-område via SharePoint-grupper 

En SharePoint-gruppe er en samling brukere som alle har det samme tillatelsessettet til områder og innhold. I stedet for å tilordne tillatelser én person om gangen, kan du bruke grupper til enkelt å tilordne det samme tillatelsesnivået til mange personer samtidig.

[Tilpasse tillatelser for SharePoint-område](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
