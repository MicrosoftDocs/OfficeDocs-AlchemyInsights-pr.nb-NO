# <a name="how-to-customize-sharepoint-site-permissions-via-sharepoint-groups"></a>Slik tilpasser du SharePoint-områdetillatelser via SharePoint-grupper 

En SharePoint-gruppe er en samling brukere som alle har samme sett med tillatelser til områder og innhold. I stedet for å tilordne tillatelser én person om gangen, kan du bruke grupper til å tilordne det samme tillatelsesnivået til mange samtidig.

[Tilpasse tillatelser for SharePoint-området](https://docs.microsoft.com/sharepoint/customize-sharepoint-site-permissions)
