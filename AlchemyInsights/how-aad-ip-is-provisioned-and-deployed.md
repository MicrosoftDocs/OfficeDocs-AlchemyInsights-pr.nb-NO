# <a name="how-azure-active-directory-identity-protection-is-provisioned-and-deployed"></a>Hvordan er beskyttelse av Azure Active Directory-identitet klargjort og distribuert?

Som standard er funksjonene i Azure Active Directory-identitet aktivert på leier nivå for alle brukere i leieren.

Hvis du vil ha mer informasjon, kan du se [Hva er Azure Active Directory-identitets beskyttelse?](https://go.microsoft.com/fwlink/?linkid=2130395)
