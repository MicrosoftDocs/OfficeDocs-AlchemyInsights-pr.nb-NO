# <a name="issues-seeing-calendar-freebusy-information"></a>Problemer som ser informasjon om ledige og opptatte tidspunkt i kalenderen

Hvis du ikke kan se informasjon om ledige og opptatte tidspunkt i kalenderen i andre Outlook-kalendere, eller hvis andre ikke kan se informasjon om ledige og opptatte tidspunkt i kalenderen, bruker du [Microsoft-assistenten for støtte og gjenoppretting for microsoft 365](https://diagnostics.office.com/).
