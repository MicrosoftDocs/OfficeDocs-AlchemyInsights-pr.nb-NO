# <a name="issues-seeing-calendar-freebusy-information"></a>Problemer med å se informasjon om ledige og opptatte tidspunkt i kalenderen

Hvis du ikke kan se informasjon om ledige og opptatte tidspunkt i andre Outlook-kalendere, eller hvis andre personer ikke kan se ledig/opptatt-informasjon i kalenderen, bruker du [Microsoft Kundestøtte og gjenoppretting Assistant for Office 365](https://diagnostics.office.com/).
