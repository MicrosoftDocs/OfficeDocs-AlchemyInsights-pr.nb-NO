# <a name="quick-edit"></a>Rask redigering

Du kan redigere et element i en SharePoint-liste på to måter. Du kan redigere ett enkelt element i listevisning, som er standardmetoden. Eller for å være enda mer produktiv, kan du redigere flere elementer i hurtigredigeringsvisning. Du må bidra nivå tillatelse eller høyere for å redigere elementer. Se [Redigere listeelementer i SharePoint Online](https://support.microsoft.com/office/dac1a1c3-a80b-4082-ba57-715cf613d0f7).
