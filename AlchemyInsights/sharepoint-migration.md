# <a name="migrate-options-to-sharepoint-online"></a>Overføre alternativer til SharePoint Online

Det finnes ulike alternativer som er tilgjengelige for å overføre innholdet til SharePoint Online, avhengig av størrelsen på og antallet filer du trenger å flytte, kan du se en liste over alternativer som [ligger her](https://docs.microsoft.com/sharepointmigration/migrate-to-sharepoint-online).

Hvis du vil ha mer informasjon om innholdsoverføring, gå til koblingene nedenfor.

- [Overføringsverktøyet i SharePoint](https://docs.microsoft.com/sharepointmigration/introducing-the-sharepoint-migration-tool)

- [SharePoint Online og ODB overføring hastighet](https://docs.microsoft.com/sharepointmigration/sharepoint-online-and-onedrive-migration-speed)

- [Migrering av data og regulere SPO-tjenesten](https://blogs.technet.microsoft.com/sposupport/2017/08/12/data-migration-and-spo-service-throttling/).


- [Unngå å få begrenset eller blokkert i SharePoint Online](https://docs.microsoft.com/sharepoint/dev/general-development/how-to-avoid-getting-throttled-or-blocked-in-sharepoint-online)

- [Vurdering av overføringsverktøyet SharePoint (SMAT)](https://www.microsoft.com/download/details.aspx?id=53598&amp;751be11f-ede8-5a0c-058c-2ee190a24fa6=True)

**Merknad**: overføringsverktøyet i SharePoint bare støtter overføring fra SharePoint 2010 og 2013. Versjon 2016 eller 2019 støttes ikke på dette tidspunktet.
