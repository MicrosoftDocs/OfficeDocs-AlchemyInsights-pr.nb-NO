# <a name="ignorelegalhold-switch"></a>**– IgnoreLegalhold-** bryter

IgnoreLegalhold som brukes i følgende unifiedgroup **,** er ikke tilgjengelig eller fungerer for Exchange Online-brukerobjekter.

- Deaktiver-post boks
- Fjern post boks
- Deaktiver-MailUser
- Fjern-MailUser
