# <a name="550-57501-access-denied-spam-abuse-detected"></a>550 5.7.501 Ingen tilgang, spammisbruk oppdaget

Denne meldingen oppstår vanligvis når brukere sender e-postmeldinger fra IP-adresser ved hjelp av det første *onmicrosoft.com* domenet som er tilordnet til nye leiere i Microsoft 365. Den enkleste måten å løse dette problemet på, er å:

1. [Legg til et domene i leieren](https://docs.microsoft.com//office365/admin/setup/add-domain).

2. [Endre brukernes primære e-postadresse](https://docs.microsoft.com//office365/admin/add-users/change-a-user-name-and-email-address) til det nye egendefinerte domenet du nettopp la til.
