# <a name="550-57705-access-denied-tenant-has-exceeded-threshold"></a>550 5.7.705 tilgang, leier har overskredet terskelen

Denne meldingen vises når for mye spam eller bulk e-post som er sendt av Office 365-organisasjon. Hvis du vil fjerne denne blokken, må du forstå og forklare årsaken til en agent for støtte, i tillegg til å løse det underliggende problemet.
