# <a name="screen-sharing-with-skype-consumer-users"></a>Skjermdeling med Skype forbruker brukere

Hvis du vil ha generelle instruksjoner for hvordan du deler programmet, lære om begrensninger med deling og feilsøke ressurser, kan du se gjennom [dele skjermen i Skype for bedrifter](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

For øyeblikket Skype for Business-brukere ikke kan dele skjermen med Skype forbruker brukere. Vennligst [Klikk her](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) for å gi tilbakemelding om aktivering skjermdeling med forbrukere Skype-brukere.
