# <a name="what-is-autosave"></a>Hva er AutoSave?

Autolagring er en funksjon som er tilgjengelig i Excel-, Word- og PowerPoint-abonnenter for Microsoft 365 som lagrer filen automatisk, med noen få sekunder, mens du arbeider. 

Hvis du vil ha vanlige spørsmål, kan du se [Hva er Autolagring?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
