# <a name="what-is-autosave"></a>Hva er automatisk lagring?

Automatisk lagring er en ny funksjon som er tilgjengelig i Microsoft Excel, Word og PowerPoint for Office 365-abonnenter som lagrer filen automatisk, hvert få sekunder, mens du arbeider. 

For vanlige spørsmål, kan du se [Hva er automatisk lagring?](https://support.office.com/article/6d6bd723-ebfd-4e40-b5f6-ae6e8088f7a5)
