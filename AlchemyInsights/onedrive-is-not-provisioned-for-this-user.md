# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive er ikke klargjort for denne brukeren

For nyopprettede kontoer må brukere logge på administrasjonssenteret for Microsoft 365 med sin egen konto og klikke OneDrive for å starte leveringen av OneDrive-området manuelt.
  
Men hvis brukeren allerede har klikket OneDrive og deres personlige område ikke er klargjort innen 24 timer, kan du logge en support billett for ytterligere assistanse.
