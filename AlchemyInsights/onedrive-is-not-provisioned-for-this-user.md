# <a name="onedrive-is-not-provisioned-for-this-user"></a>OneDrive er ikke klargjort for denne brukeren

For nylig opprettede kontoer må brukerne logge seg på administrasjons senteret for Microsoft 365 med sin egen konto og klikke OneDrive for å starte avsetningen av OneDrive-nettstedet manuelt.
  
Hvis brukeren imidlertid allerede har klikket OneDrive og det personlige nettstedet ikke er klargjort innen 24 timer, kan du logge en støtte forespørsel for å få hjelp.
