# <a name="change-language"></a>Endre språk

1.    Logg på Outlook på weben.
2. Velg **Innstillinger**øverst på siden, > **Vis alle innstillinger i Outlook**.
3. Velg **Generelt** > **språk og klokkeslett**.
4. Velg språket du vil bruke.
5. Velg **Lagre**.
 
Hvis du vil ha mer informasjon, se [Endre tidssone og språkinnstillinger i Outlook på weben](https://support.office.com/article/65239869-12e7-4a9d-bca1-76b0ad7ce273).
