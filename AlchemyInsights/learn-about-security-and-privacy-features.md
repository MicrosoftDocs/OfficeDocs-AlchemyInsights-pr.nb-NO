# <a name="learn-about-the-security-and-privacy-features-of-microsoft-search-in-bing"></a>Finn ut mer om sikkerhets- og personvernfunksjonene i Microsoft Search i Bing

Sikkerhets- og personvernfunksjonene i Microsoft Search i Bing hjelper en organisasjon med å beskytte brukerdata og arbeidsplassdata. Forespørsler gjøres for eksempel via HTTPS, og tilkoblinger krypteres fra ende til ende.

Hvis du vil ha mer informasjon, kan du se Sikkerhet [og personvern for Microsoft Søk i Bing](https://docs.microsoft.com/microsoftsearch/security-for-search).
