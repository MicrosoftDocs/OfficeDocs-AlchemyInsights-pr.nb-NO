# <a name="microsoft-bookings-app"></a>Microsoft Bestillinger-appen

Microsoft Bookings er en avtaleplanleggingsapp som hjelper deg med å holde oversikt over bestillingene dine, de ansatte og kundene dine. Hvis du vil ha informasjon eller feilsøking i Bestillinger-appen, kan du se [Bestillingsartikler](https://support.office.com/article/b9c9295c-c654-4b10-b5cc-f739825fc092).
