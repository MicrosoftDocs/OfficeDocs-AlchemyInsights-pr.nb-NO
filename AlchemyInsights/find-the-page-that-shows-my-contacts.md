# <a name="find-the-page-that-shows-my-contacts"></a>Finne siden som viser kontaktene mine

Du kan søke etter kontaktene dine på samme måte som du søker etter e-postene dine.
 
Trykk på forstørrelsesglasssøkeikonet, og skriv deretter inn kontaktsøkeforespørselen i boksen øverst på skjermen. (**Protip:** Du kan dobbelttrykke på forstørrelsesglasset for å bringe fokus til søkeboksen.) Resultatene dine vises i Kontakter-delen.
