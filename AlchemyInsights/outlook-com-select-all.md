# <a name="how-to-select-all-in-outlookcom"></a>Hvordan du "merker alle" i Outlook.com

Du kan bare velge alle hvis alle meldinger som vises. Dette er for å hindre utilsiktet handlingene på meldinger du ikke kan se.

**CTRL + klikk** velger flere individuelle meldinger.

**Skift + klikk** merker alle meldinger mellom første og siste meldingen du velger.
