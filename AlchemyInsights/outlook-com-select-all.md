# <a name="how-to-select-all-in-outlookcom"></a>Slik "velger du alle" i Outlook.com

Du kan bare velge alt hvis alle meldingene er synlige. Dette er for å forhindre utilsiktede handlinger på meldinger du ikke kan se.

**CTRL + klikk** velger flere individuelle meldinger.

**SKIFT + klikk** velger alle meldinger mellom den første og siste meldingen du velger.
