# <a name="identify-your-tenant-id"></a>Identifisere tenant-ID-en

Finn organisasjons-ID-en for Microsoft 365 (tenant-ID) i administrasjonssenteret for Azure.

1. Gå til [Katalog-ID-boksen](https://aka.ms/AzurePropertiesPage)på **Egenskaper-siden.**

2. Kopier ID-en etter behov.

Hvis du vil ha mer informasjon, kan du se [Finne organisasjons-ID-en for Microsoft 365.](https://docs.microsoft.com/onedrive/find-your-office-365-tenant-id)
