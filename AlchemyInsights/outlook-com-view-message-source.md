# <a name="how-to-view-message-headers-or-message-source"></a>Slik viser du meldingshoder eller meldingskilde

Hvis du vil se på meldingshodene og andre detaljer i en e-post, velg tre prikker på hjørnet øverst i leseruten, og velg **Vis meldingskilde**.
