# <a name="adding-external-users"></a>Legge til eksterne brukere

Trinnene for å legge til eksterne/gjeste brukere kan variere avhengig av hva du trenger for at de skal kunne få tilgang til. Velg ett av følgende:
  
- [Oversikt over ekstern deling for SharePoint Online](https://docs.microsoft.com/sharepoint/external-sharing-overview)

- [Gjeste tilgang i Microsoft 365-grupper](https://support.office.com/article/guest-access-in-office-365-groups-bfc7a840-868f-4fd6-a390-f347bf51aff6)

- [Gjeste tilgang i Microsoft Teams](https://docs.microsoft.com/microsoftteams/guest-access-checklist)
