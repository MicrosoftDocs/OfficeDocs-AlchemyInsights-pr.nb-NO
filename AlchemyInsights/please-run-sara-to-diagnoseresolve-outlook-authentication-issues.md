# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Bruk SaRA til å diagnostisere og løse problemer med Outlook-godkjenning

Vi anbefaler at du bruker [Outlook holder spør etter mitt passord](https://aka.ms/SaRA-OutlookPwdPrompt-Alchemy) Diagnostic på den berørte maskinen til å feilsøke problemer der Outlook kontinuerlig ber om et passord. Dette [SaRA](https://diagnostics.office.com/#/) diagnostiske gjør automatiserte kontroller og returnerer mulige løsninger for deg å bruke for å løse eventuelle oppdagede problemer.
