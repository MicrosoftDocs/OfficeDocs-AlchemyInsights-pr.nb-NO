# <a name="workflows-in-sharepoint"></a>Arbeidsflyter i SharePoint

<p>SharePoint-arbeidsflyter er ferdig programmerte programmer som Strømlinjeform og Automatiser et bredt spekter av forretningsprosesser. Arbeidsflyter kan variere fra samler signaturer, tilbakemelding eller godkjenning for planen eller dokumenter, til å spore gjeldende status for en rutinemessig prosedyre.</p> <p><strong>Komme i gang:</strong> Hvis du&rsquo;re bare komme i gang med SharePoint-arbeidsflyter, kan du se <a href="https://support.office.com/en-us/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3">Introduction to SharePoint-arbeidsflyter</a> for mer informasjon.</p> <p><strong>Microsoft flyt:</strong> Vurder å bruke Microsoft Flow i stedet for de klassiske arbeidsflytløsninger for SharePoint. For mer informasjon, kan du se <a href="https://support.office.com/en-us/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01">opprette en flyt for en liste eller et bibliotek i SharePoint Online- eller OneDrive</a>.</p> <p><strong>Arbeidsflyt feilsøking:</strong> Feilsøking arbeidsflyter, ligner på tidligere versjoner. For mer informasjon, se <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/debugging-sharepoint-server-workflows">Feilsøking i SharePoint-arbeidsflyter</a>. For mer informasjon om feilsøking, kan du se <a title="feilsøking i forbindelse med SharePoint Server-arbeidsflyten valideringsfeil" href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio">Feilsøking i forbindelse med feil i datavalideringen for SharePoint Server-arbeidsflyten</a>.&nbsp;</p>
