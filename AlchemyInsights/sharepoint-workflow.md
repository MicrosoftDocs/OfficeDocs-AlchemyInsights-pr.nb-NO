# <a name="workflows-in-sharepoint"></a>Arbeidsflyter i SharePoint

SharePoint-arbeidsflyter er forhåndsprogrammerte apper som effektiviserer et bredt utvalg av forretningsprosesser. Se linkene nedenfor for mer informasjon.

## <a name="getting-started"></a>Komme i gang

Hvis du nettopp har kommet i gang med SharePoint-arbeidsflyter, kan du se [Innføring i SharePoint-arbeidsflyter](https://support.office.com/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3).

## <a name="troubleshoot-and-debug-a-sharepoint-workflow"></a>Feilsøke og feilsøke en SharePoint-arbeidsflyt

Se [Feilsøke valideringsfeil for SharePoint-arbeidsflyt](https://docs.microsoft.com/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio) og [feilsøking av SharePoint-arbeidsflyter](https://docs.microsoft.com/sharepoint/dev/general-development/debugging-sharepoint-server-workflows).

## <a name="power-automate"></a>Strøm automatisere

Vurder å bruke [Power Automate](https://docs.microsoft.com/power-automate/modern-approvals) i stedet for de klassiske SharePoint-arbeidsflytløsningene.
