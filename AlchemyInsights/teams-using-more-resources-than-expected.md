# <a name="teams-is-using-more-computer-resources-than-expected"></a>Team bruker flere data maskin ressurser enn forventet

Utmerket ytelse og bruker opplevelse er de beste bekymringene hos Microsoft. Teams start tid er forbedret, og vi fortsetter å forbedre team, inkludert samlet ressurs forbruk.  

Hvis du vil ha informasjon om Teams-og ressurs bruk, kan du se [hvordan Microsoft Teams bruker minne](https://docs.microsoft.com/microsoftteams/teams-memory-usage-perf)  og  [Teams mens de er på bærbare data maskiner koblet til 4 000/HDR-skjermer](https://docs.microsoft.com/MicrosoftTeams/troubleshoot/known-issues/teams-slow-video-meetings-laptops-4k).
