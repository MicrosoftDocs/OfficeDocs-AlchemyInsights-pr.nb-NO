# <a name="manage-your-onedrive-storage"></a>Administrere lagringsplassen for OneDrive

Slik endrer du standard-lagring for nye og eksisterende brukere i OneDrive:
  
- Gå til [lagringssiden for administrasjonssenteret OneDrive](https://admin.onedrive.com/?v=StorageSettings), og skriv deretter inn et nytt tall i GB.
    
Denne storage space innstillingen gjelder for alle brukere som du ikke har angitt bestemte lagringsgrenser. Hvis du vil endre lagringsplass for bestemte brukere, må du bruke Microsoft PowerShell. For informasjon om hvordan du gjør dette, kan du se [endre brukernes OneDrive lagringsplass ved å bruke PowerShell](https://go.microsoft.com/fwlink/?linkid=866402). Hvis du vil kontrollere hvor mye lagringsplass du vil bruke i din egen OneDrive, kan du se [Manage OneDrive for bedriftslagring](https://go.microsoft.com/fwlink/?linkid=866429). For informasjon om lagring som følger med hver plan, kan du se [OneDrive for beskrivelse av tjenesten](https://go.microsoft.com/fwlink/p/?LinkID=826071).
