# <a name="sharepoint-online-permission-levels"></a>Tillatelsesnivåer for SharePoint Online

SharePoint leveres med noen standard tillatelsesnivåer som du kan bruke, for eksempel tillatelsesnivået Bidra, og Vis bare. Hvis de ikke oppfyller dine behov, kan du opprette flere tillatelsesnivåer. Det er enklest å holde oversikt over tillatelser hvis du ikke redigere eller slette standardnivåene, og hvis du legger til personer i grupper og tilordne tillatelsesnivåer til gruppene i stedet for å tilordne tillatelsesnivåer til individuelle brukere.
  
- [Forstå tillatelsesnivåer](https://go.microsoft.com/fwlink/?linkid=867071)
    
- [Hvordan du oppretter og redigerer tillatelsesnivåer](https://go.microsoft.com/fwlink/?linkid=867072)
