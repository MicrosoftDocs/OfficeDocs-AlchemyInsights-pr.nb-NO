# <a name="onboarding-microsoft-defender-atp"></a>Innlasting av Microsoft Defender ATP

Hvis du vil ha ressurser til å reserverte Microsoft Defender ATP for Windows 10, Windows-servere og Mac og Linux-maskiner, kan du se: 

- **Windows 10-maskiner** Avhengig av hvilke bedrifts krav du har, kan du velge blant alternativer for å aktivere en maskin som kjører Windows 10. Hvis du vil ha mer informasjon, kan du se [innebygde verktøy og metoder for Windows 10-enheter](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints). 

- **Windows-servere** Hvis du vil ha integrert Microsoft Defender ATP for din versjon av Windows Server, kan du se [innebygde servere i Microsoft DEFENDER ATP service](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-server-endpoints).

- **Ikke-Windows-endepunkter**  Før du har innebygd et ikke-Windows-system, må du oppfylle system kravene for [Mac](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-mac#system-requirements) eller [Linux](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-linux#system-requirements).

    Når du har bekreftet system kravene, kan du se [innebygde ikke-Windows-enheter](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints-non-windows#onboarding-non-windows-machines).
