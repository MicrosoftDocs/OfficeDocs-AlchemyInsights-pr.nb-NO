# <a name="inbox-not-loading-with-firefox"></a>Innboksen ikke lastet med Firefox

Enkelte tilleggsprogrammer i webleseren Firefox kan forhindre at innboksen Outlook.com innlasting.
  
Gå til Firefox-støtte for å lære hvordan du kan [deaktivere eller fjerne tillegg](https://support.mozilla.org/kb/disable-or-remove-add-ons).
