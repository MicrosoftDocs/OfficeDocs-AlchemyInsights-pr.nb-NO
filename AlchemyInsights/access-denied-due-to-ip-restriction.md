# <a name="access-denied-due-to-ip-restriction"></a>Ingen tilgang på grunn av IP-begrensning

Som IT-administrator kan du kontrollere tilgang til SharePoint-og OneDrive-ressurser i Microsoft 365, basert på definerte nettverks plasseringer du stoler på. Dette er også kjent som steds BAS ert policy. Hvis du vil ha mer informasjon, kan du se [kontrollere tilgang basert på nettverks plassering](https://docs.microsoft.com/sharepoint/control-access-based-on-network-location).
