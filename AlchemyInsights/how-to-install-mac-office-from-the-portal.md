# <a name="how-to-install-mac-office-from-the-portal"></a>Slik installerer du Mac Office fra portalen

1. [Avinstaller Office 2011 for Mac fra data maskinene dine](https://support.office.com/article/4bfcd230-0ea1-4656-bf30-dbfa44d358fa?wt.mc_id=Alchemy_ClientDIA).

2. [Behandle innstillinger for nedlasting av program vare](https://docs.microsoft.com/DeployOffice/manage-software-download-settings-office-365).

3. [Tilordne lisenser til brukere](https://docs.microsoft.com/microsoft-365/admin/subscriptions-and-billing/assign-licenses-to-users).

4. Brukere kan deretter installere Office-programvaren selv, eller du kan bruke [Distribuer Office til Mac-brukerne](https://docs.microsoft.com/DeployOffice/mac/deployment-guide-for-office-for-mac).
