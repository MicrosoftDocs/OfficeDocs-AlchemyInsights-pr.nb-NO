# <a name="learn-about-microsoft-365-groups"></a>Lær mer om Microsoft 365-grupper

Microsoft 365 Grupper er en tjeneste som fungerer med Office 365-verktøyene du bruker allerede, slik at du kan samarbeide med lagkameratene når du skriver dokumenter, opprette regneark, arbeide med prosjektplaner, planlegge møter eller sende e-post. Hvis du vil ha mer informasjon, kan du se [Lære om Microsoft 365 Grupper](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
