# <a name="view-your-users-microsoft-365-apps-activity"></a>Vise brukernes aktivitet i Microsoft 365-apper

[Office-aktiveringsrapporten](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) gir deg en oversikt over hvilke brukere som har installert og aktivert Microsoft 365-appene sine på minst én enhet.

I tillegg kan du få en oversikt over brukernes Microsoft 365-apper-aktivitet ved hjelp av [bruksrapporten for Microsoft 365-apper.](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide)
