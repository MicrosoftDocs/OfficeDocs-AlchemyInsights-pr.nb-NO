# <a name="set-field-values-using-parameters-passed-to-a-form---lookup-fields"></a>Angi feltverdier ved hjelp av parametere som sendes til et skjema - oppslagsfelt

Angi verdien for oppslagsfelt og Eksempel: Bruk Xrm.Utility.openEntityForm til å åpne et nytt vindu. Du kan ikke angi verdiene for partylist eller anslag.

Hvis du vil ha mer informasjon, kan du se [Angi verdier for oppslagsfelt](https://docs.microsoft.com/previous-versions/dynamicscrm-2016/developers-guide/gg334375(v=crm.8)#set-values-for-lookup-fields).
