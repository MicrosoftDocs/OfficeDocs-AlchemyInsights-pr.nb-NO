# <a name="outlook-will-now-default-block-external-content-in-smime"></a>Outlook vil nå standard blokkere eksternt innhold i S/MIME

Vi introduserer ekstra sikkerhetstiltak i Outlook for Windows ved å blokkere eksternt innhold i S/MIME-krypterte og signerte meldinger som standard og forhindre overstyringer per melding.

Hvis du vil ha mer informasjon, kan du se [Outlook for Windows blokkerer eksternt innhold i S/MIME-meldinger](https://support.office.com/article/2d3a4af1-fe41-475f-a888-fc7b997d112e).
