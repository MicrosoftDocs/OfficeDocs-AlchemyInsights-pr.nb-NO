# <a name="hybrid-configuration-wizard"></a>Veiviser for konfigurasjon av Hybrid

Veiviseren for konfigurasjon av Hybrid er en app som hjelper deg med å konfigurere en ny hybrid distribusjon mellom den lokale Exchange-organisasjonen og Microsoft 365. Hvis du allerede har konfigurert en hybrid distribusjon, hjelper veiviseren deg med å oppdatere innstillingene for hybrid distribusjon. For mer informasjon, sjekk ut:
  
- [Kjør veiviseren for konfigurasjon av Hybrid](https://technet.microsoft.com/library/mt595788%28v=exchg.150%29.aspx)

- [Veiviser for konfigurasjon av Hybrid](https://technet.microsoft.com/library/hh529921%28v=exchg.150%29.aspx)

- [Vanlige spørsmål om veiviseren for konfigurasjon av Hybrid](https://technet.microsoft.com/library/mt488940%28v=exchg.150%29.aspx)
