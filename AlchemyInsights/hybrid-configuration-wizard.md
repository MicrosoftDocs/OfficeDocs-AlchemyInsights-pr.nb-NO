# <a name="hybrid-configuration-wizard"></a>Vei viser for hybrid konfigurasjon

Vei viseren for hybrid konfigurasjon er en app som hjelper deg med å konfigurere en ny hybrid distribusjon mellom den lokale Exchange-organisasjonen og Microsoft 365. Hvis du allerede har en hybrid distribusjon, kan du bruke vei viseren til å oppdatere innstillingene for hybrid distribusjon. Hvis du vil ha mer informasjon, kan du se:
  
- [Kjøre vei viser for hybrid konfigurasjon](https://technet.microsoft.com/library/mt595788%28v=exchg.150%29.aspx)

- [Vei viser for hybrid konfigurasjon](https://technet.microsoft.com/library/hh529921%28v=exchg.150%29.aspx)

- [Vanlige spørsmål om vei viser for hybrid konfigurasjon](https://technet.microsoft.com/library/mt488940%28v=exchg.150%29.aspx)
