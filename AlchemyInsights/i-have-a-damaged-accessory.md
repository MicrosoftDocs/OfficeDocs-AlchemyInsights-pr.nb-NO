# <a name="i-have-a-damaged-microsoft-accessory"></a>Jeg har et skadet Microsoft-tilbehør

Den begrensede Microsoft Standard-garantien dekker ikke skadede, tapte eller stjålne enheter eller tilbehør. Du kan se hva som er tilgjengelig på Nettet i Microsoft Store for å erstatte enheten eller tilbehøret, eller du kan se Garantidokumentasjon for mer informasjon.

Hvis du vil finne garantiinformasjonen for enheten eller tilbehøret, kan du se Garantier, utvidede serviceplaner og vilkår [& for enheten.](https://support.microsoft.com/topic/warranties-extended-service-plans-and-terms-conditions-for-your-device-eedf7a23-84a7-1a47-480b-0e10503eedf5)

Se etter en erstatning for enheten eller tilbehøret ved å gå til [Microsoft Store](https://www.microsoft.com/) og velge Enheter-menyen.
