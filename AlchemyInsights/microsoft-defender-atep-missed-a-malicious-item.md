# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Microsoft Defender ATP gikk glipp av et skadelig element

- Send den skadelige filen til Microsoft for analyse på [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/filesubmission/). 
- Hvis du er logget på, finner du innsendingsloggen for en fil på [Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/submissionhistory).
- Hvis du vil ha mer informasjon, kan du se [Sende filer til analyse](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide).
