# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Legge til en utvidelse i Microsoft Edge fra Microsoft Edge-tillegg-butikken

Slik legger du til en utvidelse i Microsoft Edge fra Microsoft Edge-tillegg-butikken:

- Åpne Microsoft Edge.
- Velg **... > utvidelser > [få utvidelser for Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Velg filtypen du vil legge til, og velg deretter **Hent**.
- Se gjennom endringene som filtypen kan gjøre, i bekreftelses dialog boksen, og velg **Legg til utvidelse**.

Deretter får du en bekreftelse om at filtypen er lagt til.
