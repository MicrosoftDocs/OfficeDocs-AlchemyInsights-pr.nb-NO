# <a name="workflows-in-sharepoint"></a>Arbeidsflyter i SharePoint

Arbeidsflyter i SharePoint kan du utforme og automatisere forretningsprosesser. SharePoint støtter to primære arbeidsflyt utviklingsmiljøer for redigering av arbeidsflyter: SharePoint Designer og Visual Studio. 

Se [utvikle SharePoint-arbeidsflyter i SharePoint Designer og Visio](https://docs.microsoft.com/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) eller [utvikle SharePoint-arbeidsflyter ved hjelp av Visual Studio](https://docs.microsoft.com/sharepoint/dev/general-development/develop-sharepoint-workflows-using-visual-studio) til å vurdere fordelene og ulempene med hvert. 

Hvis du ønsker å finne ut om arbeidsflyter er en løsning som brukes i SharePoint-miljøet eller er på jakt etter generell informasjon om å komme i gang, kan du se [Oversikt over arbeidsflyter i SharePoint](https://docs.microsoft.com/sharepoint/dev/general-development/get-started-with-workflows-in-sharepoint#overview-of-workflows-in-sharepoint).
