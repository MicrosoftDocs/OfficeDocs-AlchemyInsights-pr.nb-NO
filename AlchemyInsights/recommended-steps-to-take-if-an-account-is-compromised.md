1. <span data-ttu-id="b5b5d-104">[Tilbakestille brukerpassordet](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) umiddelbart.</span><span class="sxs-lookup"><span data-stu-id="b5b5d-104">[Reset the user's password](https://support.office.com/article/7a5d073b-7fae-4aa5-8f96-9ecd041aba9c) immediately.</span></span> <span data-ttu-id="b5b5d-105">Ikke formidle det nye passordet til sluttbrukeren via e-post.</span><span class="sxs-lookup"><span data-stu-id="b5b5d-105">Do not communicate the new password through email to the end user.</span></span> 
    
2. <span data-ttu-id="b5b5d-106">Fjern eventuelle mistenkelige [videresendingsadresser](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) angitt på postboksnivå.</span><span class="sxs-lookup"><span data-stu-id="b5b5d-106">Remove any suspicious [forwarding addresses](https://support.office.com/article/ab5eb117-0f22-4fa7-a662-3a6bdb0add74) set at the mailbox level.</span></span> 
    
3. <span data-ttu-id="b5b5d-107">Fjern eventuelle mistenkelige [innboksregler](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) angitt i postboksen.</span><span class="sxs-lookup"><span data-stu-id="b5b5d-107">Remove any suspicious [inbox rules](https://support.office.com/article/1433E3A0-7FB0-4999-B536-50E05CB67FED) set within the mailbox.</span></span> 
    
4. <span data-ttu-id="b5b5d-108">Hvis brukeren er blokkert fra å sende e-postmeldinger, kan du [gå til Begrensede brukere for å oppheve blokkeringen av kontoen](https://protection.office.com/?hash=/restrictedusers).</span><span class="sxs-lookup"><span data-stu-id="b5b5d-108">If the user is blocked from sending email, [go to the Restricted Users to unblock the account](https://protection.office.com/?hash=/restrictedusers).</span></span> <span data-ttu-id="b5b5d-109">Når du er ferdig, kan brukeren fortsette å sende meldinger i løpet av én time.</span><span class="sxs-lookup"><span data-stu-id="b5b5d-109">Once done, you should be able to resume sending messages  within 1 hour  .</span></span>
    
5. <span data-ttu-id="b5b5d-110">Fjern brukerkontoen fra eventuelle [administrative rollegrupper](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) til du er sikker på at kontoen ikke lenger er kompromittert.</span><span class="sxs-lookup"><span data-stu-id="b5b5d-110">Remove the user account from any [administrative role groups](https://support.office.com/article/eac4d046-1afd-4f1a-85fc-8219c79e1504) until you are confident that the account is no longer compromised.</span></span> 
    
<span data-ttu-id="b5b5d-111">For å minimere et mulig databrudd eller en kompromittert konto i fremtiden anbefaler vi at du leser [artikkelen Gode fremgangsmåter for sikkerhet for Office 365](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span><span class="sxs-lookup"><span data-stu-id="b5b5d-111">To minimize the potential of a data breach or a compromised account in the future, we recommend reading our [Office 365 Security best practices article](https://support.office.com/article/9295e396-e53d-49b9-ae9b-0b5828cdedc3).</span></span>
