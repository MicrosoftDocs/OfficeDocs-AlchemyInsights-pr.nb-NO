# <a name="classification-labels"></a>Klassifisering etiketter

Med følsomhets- eller klassifiseringsetiketter kan du klassifisere og bidra til å beskytte det sensitive innholdet ditt, samtidig som du sørger for at dine ansattes produktivitet og evne til å samarbeide ikke hindres.

Hvis du vil ha mer informasjon, kan du se følgende [artikkel.](https://docs.microsoft.com/microsoft-365/compliance/sensitivity-labels)
