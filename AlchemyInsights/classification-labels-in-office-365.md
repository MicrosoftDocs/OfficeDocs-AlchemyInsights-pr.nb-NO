# <a name="classification-labels"></a>Klassifisering etiketter

Med følsomhets- eller klassifiseringsetiketter kan du klassifisere og bidra til å beskytte det sensitive innholdet ditt, samtidig som du sørger for at folks produktivitet og evne til å samarbeide ikke er hindret.

For mer informasjon vennligst se følgende [artikkel](https://docs.microsoft.com/office365/securitycompliance/sensitivity-labels).
