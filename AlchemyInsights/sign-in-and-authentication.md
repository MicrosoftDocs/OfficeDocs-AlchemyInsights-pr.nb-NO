# <a name="sign-in-and-authentication"></a>Pålogging og godkjenning

Microsoft Edge tilbyr automatiske og manuelle metoder for å logge på nettstedet. Du kan også velge en SSO for ett enkelt påloggingsalternativ. Hvis du vil lære mer om Edge-pålogging, kan du se [Hvordan brukere kan logge på Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-security-identity#how-users-can-sign-into-microsoft-edge).
