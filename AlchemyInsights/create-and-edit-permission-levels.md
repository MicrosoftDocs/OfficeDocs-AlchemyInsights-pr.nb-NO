# <a name="working-with-sharepoint-permission-levels"></a>Arbeide med tillatelsesnivåer for SharePoint

SharePoint leveres med standard tillatelsesnivåer som du kan bruke, for eksempel **bidra** og **vise bare**. Men hvis en av disse ikke oppfyller dine behov, kan du opprette et nytt tillatelsesnivå. Husk at det er enklere å holde rede på tillatelsesnivåer hvis du ikke endrer standardene.

[Opprette og redigere tillatelsesnivåer](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
