# <a name="working-with-sharepoint-permission-levels"></a>Arbeide med tilgangs nivåer i SharePoint

SharePoint leveres med standard tillatelses nivåer som du kan bruke, for eksempel bare **bidra** og **Vis**. Men hvis et av disse ikke passer dine behov, kan du opprette et nytt tillatelses nivå. Husk at det er enklere å holde rede på tillatelses nivåer hvis du ikke endrer standard innstillingene.

[Slik oppretter og redigerer du tillatelses nivåer](https://docs.microsoft.com/sharepoint/how-to-create-and-edit-permission-levels)
