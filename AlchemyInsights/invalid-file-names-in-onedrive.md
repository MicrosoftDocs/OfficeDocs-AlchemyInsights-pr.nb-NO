# <a name="invalid-file-and-folder-names-in-onedrive-and-sharepoint"></a>Ugyldig navn på filer og mapper i OneDrive og SharePoint

Disse tegnene er ikke tillatt i fil-og mappenavn " \* : \< \> ? / \ | 
  
Noen organisasjoner har ikke ennå støtte for # og % som er slått på. Hvis du vil lære hvordan du tillater disse tegnene i organisasjonen, kan du se [aktivere # og % støtte](https://go.microsoft.com/fwlink/?linkid=862611).
