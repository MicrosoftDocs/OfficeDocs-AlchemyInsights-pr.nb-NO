# <a name="migrating-to-sharepoint-online-via-migration-manager"></a>Migrere til SharePoint Online via Migration Manager

Migration Manager ligger i det moderne administrasjonssenteret for SharePoint, og veileder deg gjennom oppsettet av klientene og opprettingen av oppgavene. Du kan angi innstillinger for globalt eller aktivitetsnivå, vise fremdrift for alle aktiviteter og laste ned aggregerte sammendrags-og oppgave nivå rapporter.

* [Kom i gang med migrerings behandling](https://docs.microsoft.com/sharepointmigration/mm-get-started)
* [Oppsett av Migration Manager-klienter](https://docs.microsoft.com/sharepointmigration/mm-setup-clients)
* [Innstillinger for overførings behandling](https://docs.microsoft.com/sharepointmigration/mm-settings)
