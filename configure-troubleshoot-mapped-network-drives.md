# <a name="how-to-configure-and-troubleshoot-mapped-network-drives"></a>Slik konfigurerer og feilsøker du tilordnede nettverksstasjoner

Tilordning av et bibliotek som en nettverksstasjon er midlertidig og støttes bare via Internet Explorer. I stedet [synkroniserer du SharePoint-filer med den nye OneDrive Sync-klienten](https://support.office.com/article/6de9ede8-5b6e-4503-80b2-6190f3354a88) som inneholder [filer on-demand](https://support.office.com/article/0e6860d3-d9f3-4971-b321-7092438fb38e). Få tilgang til alle filene dine i OneDrive uten å bruke lokal lagringsplass.

* [Feilsøke tilordnede nettverksstasjoner som er koblet til SharePoint Online](https://docs.microsoft.com/sharepoint/support/administration/troubleshoot-mapped-network-drives)
